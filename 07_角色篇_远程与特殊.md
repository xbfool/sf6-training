# 第七章：角色篇 — 远程与特殊型

## 1. JP

### 角色定位
Boss 角色，远程控制/设置型 (Zoner/Setup)。全屏飞道具+地面灵力+传送+陷阱。Kakeru 使用 JP 夺得 Capcom Cup 11 冠军。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Triomphe 三重灵力弹 | 236P | 飞道具(LP/MP/HP=1/2/3发) |
| Torbalan 地面灵力 | 214P | 不同距离地面攻击 |
| Departure 传送 | 214K | LK原地/MK前传/HK后传 |
| Amnesia 陷阱 | 236K | 设置延时触发灵力 |
| Embrace | 623P | 对空灵力，OD无敌 |
| SA1 Chornobog | 236236P | 大飞道具 |
| SA2 Lovushka | 214214P | 反击超杀，有无敌 |
| SA3 Interdiction | 236236K | 全屏超杀 |

### 推荐 BnB 连招

```
基础:
2MP > 5MP xx 236LP(Triomphe)             确认接飞道具
2MK xx 214MP(Torbalan)                   蹲中脚接地面灵力
5HP xx 236K(Amnesia设置)                 重拳接陷阱

Drive Rush:
2MK xx DR > 5MP > 5HP xx 236MP           DR延伸
Anti-air 623MP > 236LP                    对空追击

高级:
2MK xx DR > 5MP > 5HP xx 236HP > SA1     超杀收尾
DI(墙贴) > 5HP xx 236HP > 214MP > SA3    DI最大伤害
Amnesia触发 > 5HP xx 236MP > 追击         陷阱配合
```

### 核心战术: 全屏控制
```
远距离: 236P(飞道具) 牵制空间
中距离: 214P(Torbalan) 不同距离控制地面
       236K(Amnesia) 设置陷阱
近距离被逼: 214K(Departure传送) 逃脱
            OD 214K 有无敌帧
```

### 起攻: Amnesia 设置
```
击倒后 > 236K(陷阱设置在对手起身位置)
→ 对手起身时陷阱触发 → 确认追击
→ 对手提前行动躲避 → JP反应惩罚
→ 双重威胁: 陷阱 + JP主动攻击
```

### 传送起攻
```
击倒后 214MK(前传) → 出现在对手身后
→ 左右二择
→ 被读到会被惩罚，需要变化
```

### 优势/劣势
- (+) 全屏控制极强；陷阱设置独特；传送灵活；OD Departure防守可靠
- (-) 近身被贴压力大；设置需要时间；体力偏低；高水平陷阱可被预判

## 2. Guile (盖尔)

### 角色定位
经典蓄力角色，Sonic Boom 控制节奏 + Flash Kick 对空。Hitbox 上表现极强（蓄力天然优势）。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Sonic Boom | [4]6P | 蓄力飞道具，核心 |
| Flash Kick | [2]8K | 蓄力对空，无敌 |
| Sonic Blade | 214P | 近距离音波刃 |
| SA1 Sonic Hurricane | [4]6[4]6P | 大飞道具超杀 |
| SA3 | 236236K | Level 3 |

### Hitbox 蓄力优势（Guile 最受益）

```
完美 Sonic Boom:
↓+← (蓄力) → 松开↓← → →+P
Hitbox: 按钮精确，蓄力时间最小化

SOCD Flash Kick:
保持按住 ↓+← → 右手拇指按 ↑+K
→ 保持蹲防姿态到最后一刻才起跳
→ 反应时间更充裕

蓄力超杀 3帧滑动:
保持 ← → 快速滑动 →←→+P
通过手指滑动完成，极快
```

### 推荐 BnB 连招

```
5MP > 2MP xx [4]6P(Sonic Boom)            核心确认
j.HK > 2HK > [2]8HK(Flash Kick)          跳入接对空技
5MP > 2MP xx DR > 5MP > 5HP xx [2]8HK     DR连招
```

### 战术: Sonic Boom 墙
```
远距离不断放 Sonic Boom → 对手必须应对:
  跳过来 → Flash Kick 对空
  Drive Rush穿过 → 你准备好反击
  等待 → 你持续积累优势

= Guile 的"围墙战术"，控制节奏
```

### 优势/劣势
- (+) Sonic Boom 远程控制顶级；Flash Kick 对空可靠；Hitbox 蓄力优势巨大
- (-) 蓄力角色走位受限；前进时失去蓄力；被贴身后蓄力困难

## 3. Dhalsim (达尔锡姆)

### 角色定位
超远距离角色。四肢伸长，全游戏最远普通技。Yoga Fire 飞道具 + Yoga Teleport 传送。操作难度全游戏最高。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Yoga Fire | 236P | 飞道具 |
| Yoga Arch | 236K | 向上抛射火焰 |
| Yoga Flame | 214P | 近距离喷火 |
| Yoga Float | 空中214K | 空中悬停 |
| Yoga Teleport | 623PPP/KKK | 前方/后方传送 |
| Drill Kick | j.2MK/HK | 空中下压踢 |
| SA2 Yoga Sunburst | 214214P | 设置型火球 |
| SA3 | 236236K | Level 3 |

### 战术: 远程控制

```
超远距离: 伸长手脚(5HP/5HK/2MK)牵制 + Yoga Fire飞道具
中距离: 不要在这里！拉远或传送逃脱
近距离: 极其危险！214P/DP/传送逃脱
空中: Yoga Float 悬停 → Drill Kick 下压 / 空中火球
```

### 基础连招
```
2LP > 2LP xx 236LP(Yoga Fire)            轻确认接火
5MP(近) > 5MK xx 236MP                   近距离确认
j.2MK(Drill) > 2LP > 5MP xx 236236P(SA1) Drill起手超杀
2MP > 5MP xx DR > 5MP > 5HP xx 214LP     DR延伸
```

### 起攻: 火焰掩护
```
击倒后 236LP(慢火球) → 火焰和Dhalsim同时压向对手
SA2 Yoga Sunburst 设置 → 火球在场时对手被迫应对
传送到身后 → 左右二择
```

### 优势/劣势
- (+) 全游戏最远普通技范围；对手难以接近；传送灵活
- (-) 操作难度最高；体力最低之一；近身极弱；伸长肢体可被打到

## 4. Blanka (布兰卡)

### 角色定位
野性角色，不可预测的移动+滚球攻击。**Blanka-chan Bomb（玩偶炸弹）** 设置。MenaRD 使用 Blanka 夺得 EVO 2025 冠军！

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Rolling Attack | [4]6P | 水平滚球突进 |
| Vertical Rolling | [2]8K | 垂直滚球对空 |
| Backstep Rolling | [4]6K | 后退再前冲 |
| Electric Thunder | 236P | 电击，近身防御 |
| Blanka-chan Bomb | 214P | 放置玩偶炸弹 |
| Wild Assault | 214K | 前冲攻击 |
| SA1 Shout of Earth | 236236P | 电击超杀 |
| SA3 | 236236K | Level 3 |

### 推荐 BnB 连招

```
基础:
2LP > 2LP > 5LP xx 236LP(Electric)      轻确认接电击
2MK xx [4]6LP(Rolling)                   蹲中脚接滚球

Drive Rush:
2MK xx DR > 5MP > 5HP xx [4]6HP(Rolling) DR接重滚球
DI(hit) > 5HP xx OD236P(OD Electric) > [4]6HP  DI高伤害

最大伤害:
2MK xx DR > 5HP xx OD236P > [4]6HP > SA3  超杀收尾
```

### Blanka-chan Bomb 战术
```
击倒后 214P(放玩偶):
→ 玩偶在对手起身位置
→ 爆炸掩护下 投/打 二择
→ 对手不管玩偶 → 被炸
→ 对手处理玩偶 → 你安全接近

中立时 214HP(远距离放):
→ 控制地面空间
→ 限制对手移动
```

### 优势/劣势
- (+) 机动性极强；Bomb设置能力；滚球突进快；电击近身可靠；交叉跳强
- (-) 连招伤害偏低；滚球被防可惩罚；依赖对手犯错；套路易被适应

## 5. Lily (莉莉)

### 角色定位
小体型角色，**Wind Stock（风灵库存）** 系统。消耗库存强化必杀技。也有指令投(360+P)。

### Wind Stock 系统
- **214K (Condor Wind)**: 积攒 1 个 Wind Stock（最多 3 个）
- 消耗 Wind Stock 可强化必杀技（不消耗 Drive Gauge！）
- 击倒后用 214K 补充库存是核心节奏

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Tomahawk Buster | 623P | 升龙对空，OD无敌 |
| Condor Spire | 214P | 突进攻击 |
| Condor Dive | 空中214K | 空中俯冲 |
| Condor Wind | 214K | 积攒Wind Stock |
| Mexican Typhoon | 360+P | 指令投 |
| SA1 Breezing Hawk | 236236K | 突进超杀 |
| SA3 Raging Typhoon | 720+P | 720超投 |

### 推荐 BnB 连招

```
基础:
2LK > 2LP xx 214LP(Condor Spire)        轻确认
5MP > 5MK xx 623MP(Tomahawk)             中拳接升龙

Wind Stock:
5MP > 5MK xx Wind 214P(强化Spire) > 5MP xx 623MP  强化突刺追击
2LK > 2LP xx Wind 623P(强化Tomahawk)               强化升龙

Drive Rush:
2MK xx DR > 5MP > 5HP xx 623HP           DR延伸
2MK xx DR > 5MP > 5MK xx SA2             DR接超杀
```

### 资源管理
```
击倒对手后决策:
选项A: 214K(Condor Wind)积攒Wind Stock — 补资源
选项B: 起攻meaty — 追加伤害/投

= Lily每次击倒都要抉择: 攒资源 vs 追伤害
```

### Hitbox 360/720
- 与 Zangief/Manon 完全相同的输入方法
- 360+P (Mexican Typhoon) / 720+P (SA3)
- 参考第一章 Hitbox 360/720 部分

### 优势/劣势
- (+) Wind Stock 不消耗Drive；有指令投；机动性好；小体型
- (-) 无强化时偏弱；短手范围小；资源管理负担；体力偏低

## 6. A.K.I.

### 角色定位
毒系角色，远程控制+设陷。毒液持续伤害 + 毒水洼控制空间 + 蛇形姿态。非传统打法。

### 毒系统
- 多个必杀技命中后给对手 **中毒状态**（持续掉血 DOT）
- **毒液水洼 (Toxic Puddle)**: 地面陷阱，对手踩到中毒
- 核心思路: 不追求一击必杀，而是毒积累总伤害

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Nightshade Pulse | 236P | 毒液攻击+留毒洼 |
| Cruel Fate | 214P | 毒爪突刺 |
| Snake Step | 214K | 低姿态蛇步+派生 |
| Sinister Slide | 236K | 低姿态滑行 |
| Orchid Spring | 623P | 对空毒喷 |
| SA1 | 236236P | 大毒液超杀 |
| SA3 | 236236K | Level 3 |

### 推荐 BnB 连招

```
2MK xx 236LP(Nightshade) + 留毒洼       安全确认+设陷
2MP > 5MP xx 214MP(Cruel Fate)           确认+中毒
214MK(Snake Step) > P > 2MP > 5MP xx 214MP  蛇步连招
2MK xx DR > 2MP > 5MP > 5HP xx 214HP    DR高伤害+中毒
```

### 战术: 毒洼控制
```
击倒后 236P → 毒洼覆盖起身位置
→ 对手起身踩毒 or 被二择
= 双重威胁

蛇步接近: 214K > 各种派生
→ 低姿态回避+出招
```

### 优势/劣势
- (+) 超远距离牵制(5HP)；毒持续伤害；毒洼空间控制；对不熟悉的对手极有效
- (-) 防御薄弱无无敌技；近身劣势；单次伤害低；连招伤害偏低
