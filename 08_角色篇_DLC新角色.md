## 1. M.Bison (维加/拜森)

### 角色定位
经典反派，蓄力角色。**Double Knee Press** 突进压制 + **Psycho Mine** 精神地雷设置 + **Head Press/Devil Reverse** 空中选项。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Double Knee Press | [4]6K | 经典突进膝踢，核心 |
| Psycho Blast | [4]6P | 精神力波飞道具 |
| Head Press | [2]8K | 飞踏对手头部 |
| → Somersault Skull Diver | 命中后按P | 追击 |
| Devil Reverse | [2]8K后按P | 改为前冲打击 |
| Psycho Mine | 214P | 设置精神地雷 |
| Shadow Rise | 214K | 翻转/传送 |
| SA1 | [4]6[4]6K | 超级膝压制 |
| SA2 | [2]8[2]8K | 有无敌 |
| SA3 Ultimate Psycho Crusher | 236236P | Level 3 |

### Psycho Mine 机制
```
214P: 打中对手 → 对手身上附着地雷
→ 延时后自动爆炸 或 特定攻击手动引爆
→ 爆炸时可追击连招
→ 角落中地雷+打击 = 双重威胁
```

### 推荐 BnB 连招

```
基础:
2LK > 2LP xx [4]6LK(Double Knee)         轻确认蓄力
5MP > 5MK xx [4]6MK                       中拳确认
2MK xx [4]6LK                             蹲中脚接膝

Drive Rush:
2MK xx DR > 5MP > 5HP xx [4]6HK           DR延伸
5MP > 5HP xx [4]6HK > [4]6LK(角落追击)    重膝接轻膝

高级:
2MK xx DR > 5MP > 5HP xx OD[4]6K > 5MP xx [4]6HK > SA1  最大伤害
DI(墙贴) > 5HP > 6HP xx [4]6HK > SA3               DI最大

Psycho Mine 配合:
5HP xx 214P(Mine) > ... > Mine爆炸 > 5HP xx [4]6HK   地雷连招
```

### Hitbox 蓄力技巧
- Double Knee [4]6K: 与 Sonic Boom 同理，← 蓄力 → →+K
- Head Press [2]8K: 与 Flash Kick 同理，↓ 蓄力 → ↑+K
- SOCD 应用: 保持蹲后蓄力(↓+←)，同时用右手按 →+K 或 ↑+K
- 蓄力超杀: [4]6[4]6K = 保持← > 快速 →←→+K

### Head Press / Devil Reverse 空中游戏
```
[2]8K: 飞到空中 → 落到对手头上
→ 对手必须防或对空
→ 命中后按P追击
→ 也可在空中按P(Devil Reverse)改变轨迹
→ 不可预测的空中选项
```

### 起攻
- **Double Knee 后**: 前步 meaty 5MP / 投
- **Psycho Mine 设置**: 击倒后 214P > 地雷+Bison 双重压制
- **Head Press**: 空中多种选项迷惑对手
- **角落**: Double Knee 连续压制，蓄力优势持续施压

### 优势/劣势
- (+) 压制力极强；Double Knee突进可靠；Psycho Mine设置；空中选项丰富
- (-) 蓄力限制灵活性；前进失去蓄力；Head Press可被对空；无普通版无敌反转

## 2. Mai Shiranui (不知火舞)

### 角色定位
SNK 客串 DLC，火焰忍术+扇子攻击。高空中机动性 + 地面/空中飞道具 + 强力连招角色carry。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| 花蝶扇 Kachousen | 236P | 地面扇子飞道具 |
| 空中花蝶扇 | j.236P | 空中释放扇子 |
| 龙炎舞 Ryuuenbu | 214P | 火焰旋转攻击 |
| 忍蜂 Shinobi Bachi | 236K | 前冲突进肘 |
| 小夜千鸟 | 214K | 空中俯冲 |
| 墙壁跳 Musasabi | [2]8K 或 j.2K | 三角跳 |
| 阳炎之舞 | 623P | 对空 |
| SA1 | 236236P | 超花蝶扇 |
| SA2 | 236236K | 超忍蜂 |
| SA3 | 214214P+K | Level 3 |

### 关键战术: b.HK Hit Confirm

来源: TheGamer Mai Guide

```
后重脚 (b.HK) → Hoshi Kujaku (Unique Move):
→ Mai 最强的 hit confirm 工具之一
→ 命中确认后接连招
→ 被防也相对安全
```

### 推荐 BnB 连招

```
基础:
2MK xx 236LP(花蝶扇)                     安全确认+飞道具
2MP > 5MP xx 214MP(龙炎舞)               ~2000 标准确认

Drive Rush:
2MK xx DR > 2MP > 5MP > 5HP xx 214HP     ~2800 主力
5MP > 5HP xx 236MK(忍蜂) > DR > 5MP > 5HP xx 214HP  忍蜂DR

跳入:
j.HP > 5HP > 5HK xx 214HP                基础跳入

空中花蝶扇:
j.236LP(空中花蝶扇) > 落地 > 2MP > 5MP xx 236MK  空扇接忍蜂

角落:
j.HP > 5HP > 5HK xx 236MK(忍蜂) > 5MP > 214HP  角落高伤害
```

### 空中花蝶扇战术
```
跳跃 > j.236LP(空中扇子):
→ 扇子飞向对手，改变跳跃的节奏
→ 对手防扇子时 Mai 安全落地
→ 扇子掩护下接近/起攻
→ 击倒后 j.236LP > 扇子覆盖起身 > 安全接近
```

### 优势/劣势
- (+) 空中机动性高跳跃轨迹多变；地面空中都有飞道具；龙炎舞稳定；忍蜂突进快
- (-) 防守端缺乏可靠无敌技；近身帧数不占优；空中行动可被对空读取

## 3. Elena (艾莲娜)

### 角色定位
卡波耶拉 (Capoeira) 踢技角色。来自 SF3 系列。标志性的 **Healing（治疗）超杀** 可以回血！资源管理独特。新手友好。

### Healing 系统（核心！）
```
SA2 Healing (214214K):
→ 消耗超必杀槽回复自身体力！
→ SF6 中唯一可以回血的角色
→ 回复量取决于槽等级

核心决策:
  有槽 → 用来打伤害(SA1/SA3)？
        → 还是用来回血(SA2)？
→ 这个决策贯穿整场比赛
```

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Scratch Wheel | 623K | 升龙式上升踢，OD无敌 |
| Spin Scythe | 236K | 旋转低踢 |
| Mallet Smash | 214P | 下压攻击 |
| Lynx Tail | 214K | 回旋踢 |
| Rhino Horn | 6HK | 前冲踢突进 |
| SA1 | 打击超杀 |
| SA2 Healing | 214214K | **回血！** |
| SA3 | Level 3 |

### 推荐 BnB 连招

```
基础:
2LK > 2LK xx 236LK(Spin Scythe)         轻确认
5MK > 2MK xx 623MK(Scratch Wheel)        中拳接升龙
2MK xx 236MK                              蹲中脚确认

Drive Rush:
5MK > 6MK xx DR > 5HP xx 236HK           DR延伸
2MK xx DR > 5MP > 5HP xx 236HK > SA1     DR超杀

跳入:
j.HK > 5HK xx 623HK                      跳入连招
```

### Healing 策略

```
领先时 (你血多对手血少):
→ 保守打法 + 击倒后 SA2 回血
→ 拉大血量差距，让对手更绝望

落后时 (你血少):
→ 如果能安全击倒对手 → SA2 回血续命
→ 如果需要一击击杀 → SA1/SA3 打伤害

"安全回血时机":
→ 击倒对手后（对手倒地时间内使用）
→ 全屏距离对手被飞道具压制时
→ 对手Drive Burnout时（无法快速接近你）
```

### 资源管理决策
```
超杀槽满 → 三个选择:
1. SA1(伤害) → 连招终结用
2. SA2(回血) → 续命用
3. SA3(最大伤害) → 击杀用

→ 没有"正确答案"，需要根据场面判断
→ 这是 Elena 的战略深度
```

### 优势/劣势
- (+) Healing回血让她持久战极强；独特踢技轨迹有回避属性；中距离踢技范围好；超杀用法灵活
- (-) 用SA2回血就失去连招超杀选项；单次伤害可能不高；资源管理要求判断力；部分招式恢复帧长
