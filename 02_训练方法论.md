# 第二章：系统训练方法论

> 来源：Steam 社区指南、Skytech Gaming、Reddit r/StreetFighter 社区总结

## 一、训练阶段总览

```
阶段1: 基础操作 (0-100小时)
  └→ 阶段2: 连招与确认 (100-300小时)
      └→ 阶段3: 对战意识 (300-600小时)
          └→ 阶段4: 高级技巧 (600+小时)
```

## 二、阶段1：基础操作（铁到银段）

### 2.1 每日热身（15分钟）

```
1. 波动拳指令 (236P) — 左右两边各 50 次不失误
2. 升龙拳指令 (623P) — 左右两边各 50 次不失误
3. 反波动指令 (214K) — 左右两边各 50 次不失误
4. 前冲 (66) — 练习 Hitbox 瞬间冲刺，连续 30 次
5. 后冲 (44) — 同上
```

**Hitbox 特别练习：**
- 斜跳攻击：9+攻击键，练习跳入角度
- SOCD 冲刺：按住前→滑过后→松开后（练到自然）
- 蓄力练习（如果你的角色需要）：按住 [4] 或 [2] 至少 40 帧

### 2.2 必学基础

1. **防守优先**: 学会防御比学会进攻更重要
   - 站防（防中段和跳入）
   - 蹲防（防下段，大多数情况默认蹲防）
   - 起身时 90% 的情况应该选择防守

2. **对空**: 每个角色都有对空招式
   - 通用: 2HP（蹲重拳）大多数角色可用
   - 专用: 升龙系角色用 623P/K
   - **练习**: 训练模式设 dummy 随机跳跃，练反应对空

3. **减少跳跃**: 新手最大的毛病是乱跳
   - 跳跃在空中无法防御，是最大的风险
   - 只在有明确目的时跳（跳飞道具、确认跳入、交叉跳等）

4. **别被逼角**: 角落是最危险的位置
   - 注意走位，保持在画面中央附近
   - 被逼角时寻找逃脱机会（Drive Reversal, 前跳, 回合间重置）

### 2.3 训练模式基础设置

- **Input Display (输入显示)**: 开启，随时检查输入是否干净
- **Frame Data Display**: 开启，了解每个招式的帧数
- **Dummy Settings**:
  - Block: 设为 "After First Hit"（训练确认）
  - Recovery: 设为 "Quick Rise"（练习起攻）
  - Guard: 交替 "Stand/Crouch"（练习高低择）

## 三、阶段2：连招与确认（银到金段）

### 3.1 连招练习方法

**步骤1: 分解连招**
```
目标连招: 2MK xx DR > 5MP > 5HP xx 623HP

分解为:
(A) 2MK xx DR — 练到 10 次连续成功
(B) DR > 5MP > 5HP — 练到 10 次连续成功
(C) 5HP xx 623HP — 练到 10 次连续成功
(D) 全连: A+B+C — 练到 10 次连续成功
(E) 左右换边: 两边各 10 次连续成功
```

**步骤2: 增加压力**
- 让 dummy 随机 block，练习 **hit confirm（命中确认）**
  - 命中 → 接连招
  - 被防 → 停止或接安全招式
- 逐步加快节奏

**步骤3: 实战转化**
- 对战中刻意使用练过的连招
- 回放中检查是否正确使用

### 3.2 每个角色必练的连招类型

1. **轻攻击确认** (Light Confirm)
   ```
   2LK > 2LP > 2LK/5LK xx 必杀技
   ```
   - 用途: 近身防守反击、起身后确认

2. **中攻击确认** (Medium Confirm)
   ```
   2MK xx 必杀技 (或 2MK xx DR > 延伸连招)
   ```
   - 用途: 最常用的牵制确认

3. **惩罚连招** (Punish Combo)
   ```
   5HP > 2HP xx 必杀技 (或更长路线)
   ```
   - 用途: 对手大失误后的最大伤害

4. **跳入连招** (Jump-in Combo)
   ```
   j.HP > 5HP > 2HP xx 必杀技
   ```
   - 用途: 成功跳入后的伤害

5. **Drive Rush 连招** (DR Combo)
   ```
   2MK xx DR > 5MP > 5HP xx 必杀技
   ```
   - 用途: 核心高伤害路线

6. **超杀连招** (Super Combo)
   ```
   确认 xx 必杀技 xx SA1/SA3
   ```
   - 用途: 有超杀槽时的最大伤害

### 3.3 Combo Trial（连招挑战）

SF6 内置了每个角色的 Combo Trial：
- **Trial 1-5**: 基础连招，必须全部掌握
- **Trial 6-10**: 中级连招，掌握其中 3-4 个
- **Trial 11+**: 高级/角色限定连招，选择性学习

### 3.4 Hit Confirm（命中确认）练习

**方法：**
1. 训练模式，dummy 设为 "Random Block"
2. 用 2MK 或 5MP 打 dummy
3. 看到 **命中（Hit）** → 接连招
4. 看到 **被防（Block）** → 不出招或接安全招

**进阶：**
- 从 2 hit confirm 开始（如 5MP>5MP，看两下再决定）
- 逐步进化到 1 hit confirm（反应更快）
- 最终目标：看到命中的瞬间就条件反射接连招

## 四、阶段3：对战意识（金到钻石段）

### 4.1 中立博弈 (Neutral Game)

中立是双方都没有明显优势时的博弈状态：

**核心技能：**
- **Walking（走位）**: 前后微移控制距离，在你角色的最佳距离战斗
- **Whiff Punish（空振惩罚）**: 对手打空 → 你用招式惩罚恢复帧
- **Anti-Air（对空）**: 反应对手跳跃 → 对空招式
- **Fireball Game（飞道具博弈）**: 波动拳系角色的远程控制

**练习方法：**
- 对战时刻意不跳跃，只用地面技
- 录像回放：观察自己在中立时的站位和选择

### 4.2 起攻 (Okizeme / Oki)

击倒对手后的压制：

**通用起攻框架：**
```
击倒对手 → 走到有利位置 → 对手起身时：
  选项A: Meaty 攻击（在对手起身帧打出攻击）
  选项B: 投技（抓对手的防御）
  选项C: Shimmy（后退骗拆投 → 惩罚连招）
  选项D: 安全跳（safe jump，被升龙防住）
```

**Meaty 练习：**
1. 训练模式，击倒 dummy
2. 设 dummy "Quick Rise + Guard All"
3. 练习在 dummy 起身的精确帧出招
4. 成功的 meaty 会显示 "Meaty" 标记

### 4.3 防守

**起身防守优先级：**
```
90% 的情况: 蹲防（默认最安全）
偶尔: 站防（防对手的中段/跳入）
稀少: 起身 DP / EX DP（高风险高回报）
更稀少: Perfect Parry（最高风险最高回报）
紧急: Drive Reversal（花资源脱身）
```

**角落防守：**
- 被逼角是最危险的情况
- 选项：Drive Reversal 推开 / 前跳逃脱 / DP 反击 / 耐心防守等对手犯错
- **不要乱按**: 角落乱动会被对手投/打二择吃大伤害

### 4.4 回放分析

**每次对战后检查：**
1. 我是怎么输的？哪个招式/情况反复吃亏？
2. 我的对空成功率如何？
3. 我有没有错过惩罚机会？
4. 我的连招掉了几次？哪个环节掉的？
5. 我的 Drive Gauge 管理如何？是否进入了 Burnout？

**改进流程：**
```
对战 → 发现问题 → 训练模式针对练习 → 再对战 → 检查是否改善
```

## 五、阶段4：高级技巧（钻石到大师段）

### 5.1 帧陷阱 (Frame Trap)

在正帧的情况下故意留一点空隙，对手按键就会被打 Counter：

```
例: 5MP (+3) → 延迟 5MP
对手按任何 4帧 以上的招式 → 被你打 Counter
```

**常用帧陷阱模式：**
- +2/+3 的招式 → 延迟出下一招
- 用于对付总是乱按的对手

### 5.2 Shimmy（诈投）

```
步骤:
1. 做出要投技的姿态（走到投技距离）
2. 快速后退一小步（退出投技范围）
3. 对手按拆投 → 投技打空 → 你用重攻击惩罚
```

### 5.3 Safe Jump（安全跳）

击倒后计算帧数，跳入攻击在对手起身时 meaty 命中：
- 如果对手防御 → 你的跳入被防，你有利继续压
- 如果对手 DP → 你落地来得及防住 DP

**关键**: 需要特定角色特定击倒后的精确时机，需要实验

### 5.4 Option Select（选择性输入）

一个输入覆盖多种情况：
```
例: 跳入 j.HP 同时按 LP+LK (投)
命中 → j.HP 连招
对手后tech → 落地后投技自动出来
```

### 5.5 Mental Stack（心理层次）

格斗游戏的高手之间是心理博弈：
```
Level 0: 无脑出招
Level 1: 我出什么最安全？
Level 2: 对手会做什么？我怎么应对？
Level 3: 对手觉得我会做什么？我反过来做
Level 4: 对手知道我会反过来做，所以我再反一层
...
```

## 六、每日训练计划模板

### 新手（每天 30-60 分钟）

| 时间 | 内容 |
|------|------|
| 10 分钟 | 指令练习：波动拳/升龙/各方向必杀技，左右各 30 次 |
| 10 分钟 | 连招练习：练 2-3 个基础 BnB，每个 10 次成功 |
| 10 分钟 | 对空练习：dummy 设随机跳，练反应对空 |
| 20-30 分钟 | 排位/对战：实战运用 |

### 中级（每天 45-90 分钟）

| 时间 | 内容 |
|------|------|
| 5 分钟 | 热身：基础指令 + Hitbox 冲刺练习 |
| 15 分钟 | 连招练习：BnB + DR 连招 + 超杀连招 |
| 10 分钟 | Hit Confirm 练习：dummy 随机 block |
| 10 分钟 | 起攻/防守练习：录制 dummy 做特定行动，练应对 |
| 30-45 分钟 | 排位/对战 |
| 5 分钟 | 回放分析：看 1-2 场回放找问题 |

### 高级（每天 60-120 分钟）

| 时间 | 内容 |
|------|------|
| 5 分钟 | 热身 |
| 15 分钟 | 角色特定技术练习（帧陷阱、safe jump、option select） |
| 10 分钟 | 对策练习：针对经常遇到的对手角色 |
| 10 分钟 | 连招优化：练习最大伤害路线和角色限定连招 |
| 45-60 分钟 | 高强度对战（排位 or 战斗中心） |
| 15 分钟 | 回放深入分析 |

## 七、心态管理

来源：Steam 社区升段指南

### 7.1 核心心态

- **享受学习过程**，不要只盯着段位
- **每次只关注改进一个方面**（对空 OR 连招 OR 防守）
- **输了不可怕**，分析为什么输，下次改善
- **连败时停下来**，休息或转去训练模式

### 7.2 避免的陷阱

- 不要在训练模式花太多时间而不打实战
- 不要在实战中只用自己最舒服的招式（要刻意练新技术）
- 不要因为段位下降就急躁（段位波动是正常的）
- 不要同时学太多角色（先精通一个）
