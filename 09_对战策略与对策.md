## 一、通用防守策略

来源：Steam 社区讨论、Reddit r/StreetFighter、GigaBrain 防守指南

### 1.1 起身防守黄金法则

```
起身时 90% 的情况: 蹲防（默认最安全）

原因:
- 蹲防可以防住大多数攻击（下段+中段普通技）
- 只有 overhead(中段特殊技)/跳入 需要站防
- overhead 一般比较慢，可以反应切换

唯一需要注意: 投技（蹲防防不住投）
→ 偶尔起身拆投 或 后跳逃脱投
```

### 1.2 Drive System 防守选项

| 选项 | 指令 | 风险 | 回报 | 使用时机 |
|------|------|------|------|----------|
| Drive Reversal | 防中 6+HP+HK | 低(2格) | 推开对手 | 被持续压制时 |
| Drive Parry | MP+MK | 中(持续耗) | 安全防御 | 对手多段压制 |
| Perfect Parry | 精确MP+MK | 高(2帧窗口) | 巨大有利帧 | 读心对手出招 |
| Drive Impact | HP+HK | 中高(1格) | 反转攻势/墙贴 | 对手连续压制时 |

### 1.3 反飞道具策略

```
全屏距离:
→ 前走+防御逐步接近
→ Drive Parry 弹反（恢复Drive Gauge）
→ 跳跃（全屏跳相对安全）
→ 自己的飞道具对抗

中远距离:
→ Drive Parry 弹反后 DR 前冲
→ 预判跳（但小心对手升龙）
→ Drive Rush 穿过

近距离:
→ 不要跳！对手等你跳就升龙
→ Drive Parry > Perfect Parry
→ 无敌技穿过（如果有的话）

特殊方法:
→ 部分角色有穿弹技 (如 Cammy 的 Quick Spin Knuckle)
→ 部分角色有反射/吸收技
→ DI 可以吸收飞道具（2次armor）然后继续前进
```

### 1.4 角落防守

```
被逼角后选项:
1. 耐心防守 — 等对手犯错后惩罚
2. Drive Reversal — 花2格推开对手（最安全的脱身）
3. 前跳 — 跳到对手身后（但被读到会被对空）
4. DP/无敌技 — 高风险翻身（被防会被最大惩罚）
5. DI — 如果对手连续压制，DI可以翻转
6. Perfect Parry — 最难但回报最高

不要做的事:
× 乱按键 → 被帧陷阱打 Counter
× 乱跳 → 被对空
× 恐慌 → 保持冷静，等机会
```

## 二、进攻策略

### 2.1 中立博弈 (Footsies)

```
核心原则:
1. 控制"你角色的最佳距离"
   - 速攻角色: 尽量贴身
   - 波动拳角色: 中距离
   - 远程角色: 远距离

2. 前后微移 (Walking)
   → 引诱对手出招 → 空振惩罚
   → 进入自己的最佳距离 → 牵制技确认

3. Whiff Punish (空振惩罚)
   → 对手招式打空 → 你用 2MK 或 5HP 惩罚恢复帧
   → 需要保持在对手招式尖端的距离

4. Anti-Air (对空)
   → 对手跳 → 你用对空技
   → 对空成功 → 进入有利位置
   → 练到"看到跳就自动对空"的反应
```

### 2.2 起攻 (Okizeme) 通用框架

```
击倒对手后:
Step 1: 走到有利位置（对手起身时你在攻击范围内）
Step 2: 选择选项:

选项A: Meaty 攻击
  → 在对手起身帧精确打出攻击
  → 对手防 = 你有利继续压
  → 对手按键 = Counter Hit 大连招
  → 对手投 = 你的攻击更快先打到

选项B: 投技
  → 对手防守就被投
  → 近距离投技 / 指令投

选项C: Shimmy
  → 走到投距离 → 微后退
  → 对手拆投打空 → 你5HP惩罚连招

选项D: Safe Jump (安全跳)
  → 精确时机跳入
  → 对手防 = 你有利
  → 对手DP = 你落地来得及防

选项E: DR meaty
  → Drive Rush 贴身 → 延迟打击/投二择
  → 消耗3格Drive但非常强

= 对手必须猜！猜错吃大伤害
```

### 2.3 压制 (Pressure) / 连续攻势

```
基础压制串:
5MP(+2) > 5MP(+2) > 2MK xx 必杀技    中拳连续打点
5LP(+3) > 5LP(+3) > 2MK xx 必杀技    轻拳快速压制

帧陷阱:
5MP(+2) > 微延迟 > 5MP
→ 对手按 4帧+ 的招式就被 Counter

投/打二择:
走近 > 投(LP+LK)  或  走近 > 2MK xx 确认
→ 对手防 = 被投
→ 对手拆投 = 被2MK打中

Drive Rush 压制:
2MK xx DR(+4额外) > 5MP > 投/打
→ DR给的有利帧让整个压制串更安全

Stagger Pressure (间隔压制):
5MP > 等 > 5MP > 等 > 投
→ 故意留间隔，诱导对手按键 → 惩罚
```

## 三、角色类型对策

### 3.1 对付投技角色 (Zangief/Manon/Lily)

```
核心: 保持距离！不要被贴身

策略:
1. 控制中远距离，用牵制技打点
2. 对手 DR 接近时 → DI 反制 或 后跳
3. 对手近身后 → 不要怕投，偶尔跳跃逃脱
4. 投技读心 → 后跳 j.攻击 惩罚
5. Drive Reversal 推开

注意:
× 不要在角落被逼住 → 投技角色角落最可怕
× 不要习惯性蹲防 → 会一直被投
√ 偶尔跳 / 后跳 / 垂直跳 → 投技打空你惩罚
```

### 3.2 对付速攻角色 (Cammy/Kimberly/Rashid)

```
核心: 稳住防守，找机会惩罚

策略:
1. 蹲防为主，反应切换站防（防跳入/中段）
2. 对空要准！速攻角色经常跳
3. 被压制时 Drive Reversal 脱身
4. 对手连续压制时找间隔 → 按轻攻击打断
5. Perfect Parry 对手的固定套路

注意:
× 不要被节奏带着走
× 不要因为被压就恐慌乱动
√ 冷静防守 → 等对手出错 → 最大惩罚
```

### 3.3 对付远程角色 (JP/Dhalsim/Guile)

```
核心: 想办法接近！

策略:
1. 前走+Drive Parry 飞道具接近
2. 全屏跳跃（安全距离可以跳）
3. Drive Rush 穿过飞道具间隔
4. 接近后不要轻易被推走
5. 贴身后猛攻，不给对手重新拉开的机会

注意:
× 不要在全屏傻站被打
× 近距离不要跳（会被对空/升龙/Embrace等）
√ 耐心接近 → 一旦贴身就全力输出
√ DI 在接近过程中可以吸收飞道具
```

### 3.4 对付重型角色 (Marisa/Honda)

```
核心: 不要硬碰硬

策略:
1. 利用速度优势，快进快出
2. 不要和 Marisa 比 armor — 用投技/多段攻击破armor
3. Honda 头槌: 反应防御后惩罚（LP版-2可能惩罚不了，HP版可以大惩罚）
4. 利用飞道具控制距离
5. 中距离牵制技打点，不给他们接近的机会

注意:
× 不要被 armor 吓到继续出招
× 不要在角落和重型角色打消耗战
√ 灵活走位 → 控制距离 → 快打快走
```

## 四、确认反击 (Punish) 指南

### 4.1 通用可惩罚招式

```
大惩罚 (-10以上):
→ 被防的升龙拳 (DP) — 最常见
→ 被防的跳跃特殊技
→ 被防的超杀

中惩罚 (-6 to -9):
→ 被防的重必杀技
→ 被防的 HP/HK 等重攻击

小惩罚 (-4 to -5):
→ 用轻攻击 > 确认连招

基本安全 (-3 以内):
→ 大多数情况惩罚不了
→ 但可以抢帧(你的5LP是4帧，对方-4就可以打)
```

### 4.2 经典可惩罚场景

| 场景 | 你应该做什么 |
|------|-------------|
| 对手升龙被防 | 站起来 > 5HP > 最大伤害惩罚连招 |
| 对手 DI 被防/被反DI | 5HP > 最大伤害 |
| 对手 Burnout 被DI墙贴 | 5HP > 最大伤害（对手眩晕） |
| 对手超杀被防 | 根据帧数，5HP 或 快速惩罚 |
| 对手跳被你对空 | 对空技 > 可能接追击 |
| 对手投技打空 | 5HP > 最大伤害 |
| 对手 DR 被你防住 | 快速小技打断 或 DI 反制 |

### 4.3 Perfect Parry 后续

```
Perfect Parry 成功后:
→ 时间暂停效果
→ 你获得巨大有利帧
→ 接最大伤害惩罚连招

练习方法:
1. 训练模式设 dummy 重复某个招式
2. 练习 MP+MK 的精确时机 (2帧窗口)
3. 先练反应最快的招式 (如对手的5MP)
4. 逐步扩展到其他招式和多段攻击
```

## 五、Drive Gauge 管理策略

### 5.1 何时花费

```
值得花费:
√ DR连招确认 (3格) — 确保命中后花
√ OD必杀技确认连招 (2格)
√ 生死关头的 DI (1格)
√ Drive Reversal 脱离角落危机 (2格)

不值得花费:
× 中立时随意DR接近（可能被防后白花3格）
× 非确认情况下OD必杀技
× 被压制时恐慌DI（可能被读心惩罚）
× 长时间按住Parry（持续消耗）
```

### 5.2 Burnout 管理

```
Drive Gauge 剩 2 格以下时:
→ 减少 Drive 行动
→ 不要再用 DR 连招
→ 保守打法等待恢复
→ 前走可以加速恢复

Burnout 后:
→ 极其危险状态
→ 防守时直接扣血
→ DI 墙贴 = 眩晕 = 死
→ 约 20 秒恢复
→ 尽量跑/回避/使用SA

对手 Burnout 时:
→ 你的大好机会！
→ 积极DI（墙贴=眩晕=最大伤害）
→ 持续压制不给喘息
→ 飞道具chip damage直接扣血
```

## 六、常用术语速查

| 术语 | 含义 |
|------|------|
| BnB | Bread and Butter，日常最常用的连招 |
| Meaty | 在对手起身帧精确打出的攻击 |
| Oki | Okizeme，击倒后的起攻 |
| Shimmy | 后退骗拆投后惩罚 |
| Frame Trap | 帧陷阱，留微小间隔让对手按键被打Counter |
| Tick Throw | 轻攻击后接投技 |
| Safe Jump | 精确跳入，被DP也能防住 |
| Whiff Punish | 空振惩罚 |
| Footsies | 中距离走位博弈 |
| Zoning | 远程控制/牵制 |
| Rushdown | 近身猛攻打法 |
| Mixup | 二择/多择 |
| Reset | 中断连招制造择 |
| Tech | 拆投 (Throw Tech) |
| Chip | 防御时的微量伤害 |
| Plus/Minus | 正帧/负帧 (有利/不利) |
| Cancel | 取消，招式动画中接其他招式 |
| Buffer | 缓冲，提前输入指令 |
| Reversal | 最快时机出招（起身后/防御后） |
| Stagger | 间隔压制，不连续出招留gap |
