# 第一章：SF6 系统机制与 Hitbox 技巧

---

## 一、Drive 系统详解

SF6 的核心系统是 **Drive System（驱动系统）**，每个角色都有 **6 格 Drive Gauge**。

### 1.1 Drive Impact (DI) — 消耗 1 格

- **指令**: HP+HK
- **效果**: 超级装甲突进打击，可吸收 **2 次攻击**
- **墙角命中**: 造成 **Wall Splat（墙贴）**，可接大伤害连招
- **对方 Burnout 时墙贴**: 造成 **Stun（眩晕）**，可接最大伤害
- **被 DI 打中**: 可以用自己的 DI 反打（DI 反制 DI）
- **关键**: 不要滥用，被 Parry 或被读心跳过会被大惩罚

### 1.2 Drive Parry — 消耗持续消耗（约 0.5 格/秒）

- **指令**: 按住 MP+MK
- **效果**: 自动防御一切攻击（包括多段），恢复 Drive Gauge
- **Perfect Parry（完美弹反）**: 在攻击命中前 **2帧** 内按下 MP+MK
  - 效果：时间暂停，获得巨大有利帧，可接最大惩罚连招
  - 练习方法：训练模式中让 dummy 重复同一招，练习精确时机
- **用途**:
  - 对付飞道具（弹反波动拳）
  - 读心弹反对手的压制
  - 起身时 Perfect Parry 对手的 meaty

### 1.3 Drive Rush (DR) — 消耗 1~3 格

- **从 Parry 取消**: MP+MK → 66 — 消耗 **1 格**
- **从普通技/必杀技取消**: 连招中 66 — 消耗 **3 格**
- **效果**: 快速前冲，普通技获得 **+4帧额外有利帧**
- **核心用途**:
  - 延伸连招（最重要的连招工具）
  - 从安全的牵制技取消接近对手
  - Parry 后快速反击
- **标准 DR 连招结构**: `2MK xx DR > 5MP > 5HP xx 必杀技`

### 1.4 Drive Reversal — 消耗 2 格

- **指令**: 防御中 6+HP+HK
- **效果**: 防御反击，推开对手，获得距离
- **用途**: 被压制时脱身，对付持续压制的对手
- **注意**: 伤害很低（500），主要用于脱压而非伤害

### 1.5 Overdrive (OD) — 消耗 2 格

- **指令**: 按两个相同按键（如 PP 或 KK）使出必杀技
- **效果**: 必杀技强化版本，通常有更好的属性：
  - 更多伤害
  - 更多无敌帧
  - 更好的帧数据
  - 额外的打击效果
- **关键**: 合理管理 Drive Gauge，不要过度使用 OD 导致 Burnout

### 1.6 Burnout（驱动耗尽）

- **触发**: Drive Gauge 完全消耗完毕
- **惩罚**:
  - 无法使用任何 Drive 行动
  - 防御时直接扣血（chip damage 直接扣 HP）
  - 被 DI 墙贴会造成 **眩晕**（致命！）
  - Gauge 恢复需要约 **20 秒**
- **核心原则**: **永远避免 Burnout**，这是 SF6 中最危险的状态

### 1.7 Drive Gauge 管理要点

| 行动 | 消耗 | 备注 |
|------|------|------|
| Drive Impact | 1 格 | 廉价但风险高 |
| Drive Parry | 持续消耗 | 不要一直按着 |
| Drive Rush (Parry取消) | 1 格 | 性价比最高 |
| Drive Rush (普通技取消) | 3 格 | 核心连招工具 |
| Drive Reversal | 2 格 | 防守脱身用 |
| Overdrive | 2 格 | 强化必杀技 |

**恢复方式**: 前走、成功攻击、等待自然恢复

## 二、帧数据基础

### 2.1 什么是帧？

- SF6 运行在 **60fps**，即每秒 60 帧
- 1 帧 = 1/60 秒 ≈ 16.67 毫秒
- 所有招式都由三个阶段组成：
  - **Startup（发生帧）**: 出招到攻击判定出现的帧数
  - **Active（持续帧）**: 攻击判定存在的帧数
  - **Recovery（恢复帧）**: 攻击结束后角色无法行动的帧数

### 2.2 有利帧 / 不利帧 (Frame Advantage)

- **正帧 (+)**: 你比对手更早恢复行动。例如 +3 = 你早 3 帧行动
- **负帧 (-)**: 对手比你更早恢复行动。例如 -7 = 对手早 7 帧行动
- **0 帧**: 同时恢复

### 2.3 实战意义

| 情况 | 说明 |
|------|------|
| **+1 到 +3** | 轻微有利，可以继续压制（但对手轻攻击可能打断） |
| **+4 以上** | 大有利，可以安全继续进攻 |
| **0 到 -3** | 基本安全，对手大多数招式惩罚不到 |
| **-4 到 -6** | 不安全，可被快速招式惩罚 |
| **-7 以上** | 大不利，可被重攻击惩罚 |

### 2.4 确认反击 (Punish / Whiff Punish)

- **Punish（惩罚）**: 对手出招后处于负帧时，你用招式反击
  - 例：对手升龙被防 (-47帧) → 你出最大伤害连招
- **Whiff Punish（空振惩罚）**: 对手招式打空，你在恢复帧中反击
- **Punish Counter (PC)**: 在对手招式恢复帧中命中，触发特殊效果（额外浮空/有利帧）

### 2.5 Cancel（取消）

- **普通取消 (Normal Cancel)**: 普通技命中/被防时取消为必杀技
  - 例：`2MK xx 236P`（蹲中脚取消波动拳）
- **Special Cancel**: 必杀技取消为超必杀
  - 例：`623HP xx 236236P`（升龙取消 SA1）
- **Drive Rush Cancel**: 普通技/必杀技取消为 Drive Rush
  - 例：`2MK xx DR`（蹲中脚取消 Drive Rush）

## 三、Hitbox 控制器详解

### 3.1 什么是 SOCD？

**SOCD = Simultaneous Opposing Cardinal Directions（同时按下相反方向键）**

Hitbox 用按钮代替摇杆，可以同时按下相反方向。为了比赛合法，需要 SOCD 清洗。

来源：[Hit Box Arcade - Introduction to SOCD](https://www.hitboxarcade.com/blogs/support/what-is-socd)

### 3.2 SOCD 清洗类型

| 类型 | 效果 | 适用 |
|------|------|------|
| **中立清洗 (Neutral)** | 左+右 = 中立，上+下 = 中立 | **SF6/铁拳8 官方标准** |
| **Up Priority** | 左+右 = 中立，上+下 = 上 | 老版 Hit Box 默认 |
| **Last Input Priority** | 最后按下的方向获胜 | 部分自定义控制器 |

**SF6 的规则**: 游戏端使用 **中立清洗**（左+右=中立，上+下=中立）

### 3.3 Hitbox 瞬间冲刺 (Instant Dash)

来源：[Hit Box Arcade - SF6 Instant Dashing](https://www.hitboxarcade.com/blogs/hit-box/sf6-hb-instant-dashing)

#### 方法1：双指滑动 Dash（Double-Tap Dash）
```
步骤：
1. 中指滑过前进按钮 →
2. 按钮自然回弹到中立
3. 食指滑过同一按钮 →
结果：最快 3 帧完成 dash
```
- **优势**: 两根手指交替，比一根手指双击快得多
- **建议**: 中指先滑，食指跟上，形成肌肉记忆

#### 方法2：SOCD Instant Dash（推荐）
```
步骤：
1. 按住前进方向（如按住 →）
2. 用另一根手指滑过反方向（← ），产生 SOCD 中立
3. 松开反方向，恢复到 →
结果：4-5 帧完成 dash，更稳定
```
- **原理**: 前+后 = 中立（SOCD），松开后 = 前。等同于 前→中立→前
- **优势**: 比双击更稳定，不容易失误
- **核心建议**: 追求稳定的 4-5 帧，不要贪 2-3 帧极限

### 3.4 Hitbox 蓄力技巧

来源：[Hit Box Arcade - SF6 Charge Motions](https://www.hitboxarcade.com/blogs/hit-box/sf6-hb-charge-motions)

#### Sonic Boom (水平蓄力 [4]6+P)
```
输入：
1. 按住 ↓+← (下后) 蓄力
2. 松开 ↓ 和 ←
3. 按 →+拳
```
- **Hitbox 优势**: 按钮精确，蓄力时间最小化
- **进阶**: "Half Position" — 用右手食指跨区域按 →+P，提高速度

#### Flash Kick (垂直蓄力 [2]8+K)
```
SOCD 方法：
1. 按住 ↓+← 蓄力
2. 保持按住 ↓+←
3. 用右手大拇指同时按 ↑+脚
   → 上+下 = 中立（SOCD），但因为还按着 ←，输出是 ← + ↑ 的过渡
   → 游戏判定为有效的 [2]8 输入
```
- **关键**: 用右手大拇指同时按 ↑+脚，实现"低Flash Kick"
- **优势**: 保持下蹲姿态直到最后一刻才起跳，反应时间更充裕

#### 蓄力超杀 ([4]6[4]6+P)
```
滑动方法：
1. 保持蓄力 (←)
2. 快速: → ← → + 拳
通过手指在按钮间滑动完成，配合 Half Position 可在 3 帧内完成
```

### 3.5 Hitbox 360/720 指令（投技角色核心）

来源：[Hit Box Arcade - The Zangief Bible 360/720](https://www.hitboxarcade.com/blogs/hit-box/the-zangief-bible-360-720)

#### 360度 (SPD / 指令投) 输入法

**核心原理**: SF6 的 360 只需要 4 个主方向（↑↓←→），不需要对角线，不需要完整一圈。

**方法1 — 标准滑动（最常用）**
```
→ ↓ ← ↑+P（右 下 左 上+拳）
用食指从右滑到左经过下，最后拇指按上+拳
```

**方法2 — 从蹲防开始（避免跳跃，推荐）**
```
↓+← (蓄力/蹲防) → ← → ↑+P
已经按着下后，只需：松开下 → 按右 → 按上+拳
```

**方法3 — 垫步缓冲（实战最常用）**
```
前后走动时: → ← 已经输入了两个方向
只需补 ↓ + ↑+P 完成
```

**方法4 — Tick Throw 缓冲**
```
2LP（蹲轻拳，已经在按↓）
→ 恢复帧中输入: → ↑ ← + P
= 完成 360
```

#### 720度 (SA3 超投) 输入法

**滑动 720**
```
用食指从 → 滑到 ← 经过 ↓，再滑回来
→↓←↑→↓←↑+P
关键: ↑(跳跃键) 必须极快地点按释放（单帧操作）
```

**冲刺 720**
```
在前冲动画中隐藏 720 输入：
66(前冲) → 冲刺中输入 ↓←↑→↓←↑+P
前冲动画掩盖了 720 的输入，避免意外跳起
```

**连招缓冲 720**
```
普通技出招 → 恢复帧中输入 720 → 完成时按 P
例: 5MP出招 → 恢复帧中转两圈 → SA3 发动
```

**核心要点**: 跳跃键要当成 "POP" — 快速点按立刻松开，借反弹力抬拇指。需要大量练习！

#### 360/720 快速参考表

| 指令 | 推荐输入 | 场景 |
|------|----------|------|
| 站立 SPD | 6>2>4>8+P | 对手近身时 |
| 蹲防 SPD | (蓄1)松开>6>8>4+P | 防守后反击 |
| Tick SPD | 2LP > 6>8>4+P | 小技后缓冲 |
| 前冲 SPD | 66中 > 2>4>8+P | Drive Rush 接投 |
| 720 | 冲刺中 6>2>4>8>6>2>4>8+P | 超投 |

### 3.6 Hitbox 的整体优势与注意事项

**优势：**
- 瞬间回中立（松开按键即可，摇杆需要回正时间）
- 精确的方向输入（按钮 vs 模拟摇杆）
- 蓄力角色天然优势（多指独立控制）
- 360/720 输入更快更精确
- SOCD 技巧提供独特的输入优化

**需要额外练习的地方：**
- 空中指令（如空中波动拳）需要适应左手位置
- 走位（前后微移）初期可能不如摇杆直观
- 部分半圆指令（如 41236）需要练习手指协调
- 跳跃方向控制（特别是斜跳）需要两指配合

**常见错误：**
- 360 输入时意外跳跃 → 确保上键是最后一个输入+同时按攻击键
- 蓄力不足就出招 → 注意蓄力时间（大约 40-50 帧）
- SOCD 冲刺时按键重叠 → 练习干净的松开-按下节奏
