## 一、绿冲 (Drive Rush Cancel) 专项攻克

来源：[Steam 社区讨论](https://steamcommunity.com/app/1364780/discussions/0/3809531491905236522/)、[DR Cancel 技巧帖](https://steamcommunity.com/app/1364780/discussions/0/3815158655064787145/)、[Destructoid DR 指南](https://www.destructoid.com/street-fighter-6-guide-drive-rush-and-how-to-use-it/)

### 1.1 绿冲的两种输入方式

```
方式A（经典）: 普通技命中时 → 66（前前）
方式B（替代，2023.7月更新后）: 普通技命中时 → MP+MK

两种都能触发 Drive Rush Cancel，消耗 3 格 Drive Gauge
```

**在 Hitbox 上：**
- **方式A (66)**: 用中指按 →，松开，食指按 → （双指交替，和瞬间冲刺相同）
- **方式B (MP+MK)**: 右手直接按 MP+MK（更简单！推荐先用这个练）

### 1.2 绿冲为什么难？根本原因

```
难点1: 时机 — 必须在普通技的"取消窗口"内输入66或MP+MK
难点2: 方向污染 — 蹲中脚(2MK)时按着↓，66需要干净的→→，↓→不行
难点3: 从蹲姿到前冲 — 手指需要从↓切换到→→，动作不连贯
```

### 1.3 Hitbox 上的绿冲输入法（从简到难）

#### 方法①：MP+MK 取消法（最简单，先用这个！）

```
操作流程:
1. 按 ↓ + MK（蹲中脚出招）
2. 蹲中脚命中的瞬间 → 右手按 MP+MK
3. 角色自动绿冲前冲

输入时间线:
  ↓+MK .............. MP+MK
  [出招]            [取消窗口内]

Hitbox 优势: 左手管方向(↓)，右手管按键(MK → MP+MK)，互不干扰！
```

**练习步骤：**
```
Step 1: 训练模式，dummy 站着不动
  → 2MK > MP+MK — 反复练到绿冲稳定出来 (目标: 10/10)

Step 2: 按节奏
  → 2MK(嗒) ... MP+MK(嗒) — 像打节拍
  → 不要太快也不要太慢，找到"甜蜜点"
  → 看屏幕: 绿冲出来了 = 时机对了

Step 3: 接后续
  → 2MK > MP+MK(绿冲) > 5MP — 绿冲出来后按中拳
  → 先不管连招，只确保绿冲+一个后续稳定
```

#### 方法②：66 取消法（传统方法，更快但更难）

```
操作流程:
1. 左手按 ↓+MK（蹲中脚）
2. 松开 ↓（回中立）
3. 快速 →→（双击前）— 中指→食指 或 SOCD法

关键: ↓ 必须先松开！↓→ = ↘（斜前下），不是干净的 → ！
     至少要有 1帧 的中立/纯→ 输入

输入时间线:
  ↓+MK ... 松↓ ... →→
  [出招]   [1帧]  [取消]

Hitbox 精确输入:
  左手: 按↓(蹲) → 松↓(中立) → 不按任何方向
  右手: 按MK ..... 不按 ...... 不按
  然后:
  左手: 按→(前) → 松→(中立) → 按→(前)  ← 这就是66
```

**练习步骤：**
```
Step 1: 先练 "松↓ → →→" 的手指动作（不按攻击键）
  → 蹲着 → 松开↓ → 前冲(66)
  → 反复练到流畅

Step 2: 加上 2MK
  → ↓+MK → 松↓ → →→
  → 目标: 10/10 稳定绿冲

Step 3: 加后续
  → 2MK → 松↓ → →→ > 5MP > 5HP xx 必杀技
```

#### 方法③：SOCD 辅助法（Hitbox 专属黑科技）

```
原理: 按住→的同时按←会产生SOCD中立，松开←恢复→
可以用这个特性简化66输入

操作:
1. ↓+MK（蹲中脚）
2. 松开 ↓，同时按 →（开始前走）
3. 保持按住 →，快速点按 ←（产生SOCD中立）
4. 松开 ← → 恢复到 →
= 系统读到: → 中立 → = 66！

优势: →一直按着，只需要点一下←就完成66
      手指动作更小，更不容易失误
```

### 1.4 绿冲练不好的常见问题排查

```
问题1: 绿冲没出来（角色没有前冲）
  原因A: 66 不够快 → 两次→之间间隔太长
  原因B: 方向不干净 → ↓没松开，输入了↘而非→
  原因C: 时机太晚 → 超出了取消窗口
  解决: 开启 Input Display，检查方向输入是否干净

问题2: 绿冲出来了但后续接不上
  原因A: 绿冲后按键太慢 → DR有效时间有限
  原因B: 绿冲后距离太远 → 需要用射程更远的普通技
  解决: 绿冲后第一下用 5MP（快且远），不要用5HP

问题3: 变成了跳跃或其他招式
  原因: 方向输入混乱，可能按到了↑或斜方向
  解决: Hitbox上确保只按→，不要误触↑

问题4: 波动拳取消绿冲 (2MK xx 236P xx DR) 更难
  原因: 波动拳指令(236P)结束时你在→方向，需要回中立再→→
  解决: 236P 输入完后 → 保持按→ → 松→ → 再按→ = 完成
        或: 236P 完后直接 MP+MK（替代输入法，推荐！）
```

### 1.5 隐藏技巧：Negative Edge（负边缘）

来源：[Steam帖子](https://steamcommunity.com/app/1364780/discussions/0/3815158655064787145/)

```
设置: 多人菜单 → 设置 → 战斗控制 → 打开"按钮释放输入"

效果: 松开按键时也能触发取消
→ 社区报告 DR Cancel 从 1/5 成功率提升到 5/5

注意: 可能影响其他招式的出招，试试看是否适合你
```

## 二、Ken 迅雷脚 (Jinrai Kick) 专项攻克

来源：[EventHubs Ken Jinrai Kick](https://www.eventhubs.com/moves/sf6/ken/move/special-moves/jinrai-kick/)、[SF Wiki Jinrai Kick](https://streetfighter.fandom.com/wiki/Jinrai_Kick)、[Ultimate Frame Data Ken](https://ultimateframedata.com/sf6/ken)

### 2.1 迅雷脚系统详解

```
迅雷脚 Jinrai Kick: 236K

三种派生:
┌─ 236K > K（风鎌真蹴 Kazekama Shin Kick）
│   → 下段踢！-5帧，相对安全
│   → 最常用的派生
│
├─ 236K > 6K（業来斧刃脚 Gorai Axe Kick）
│   → 中段！overhead！但出招慢
│   → 偶尔使用的二择选项
│
└─ 236K > 4K（旋火閃刻脚 Senka Snap Kick）
    → 中高段旋踢
    → 连招扩展用

迅雷脚本体帧数:
  LK版(236LK): 发生13帧, 防御-11(不安全!)
  MK版(236MK): 发生17帧, 防御-6(被惩罚)
  HK版(236HK): 发生25帧, 防御-2(基本安全)
  OD版(236KK): 发生15帧, 防御-2(安全) + 可Drive Rush取消！
```

### 2.2 迅雷脚为什么用不好？常见问题

```
问题1: 迅雷脚出不来 / 变成别的招
  原因: 236K 输入不干净，可能变成 26K(无效) 或 23K(无效)
  解决: 开 Input Display，确认 ↓↘→+K 三个方向都有
  Hitbox: ↓ → ↓+→ → → + K，注意 ↘ 必须经过

问题2: 派生按不出来
  原因: 派生K的时机不对 — 太快或太慢
  解决: 迅雷脚本体命中后有一个"派生窗口"
        → 在窗口内按 K/6K/4K
        → 不要在236K输入时就连带按了第二个K

问题3: 派生选错了
  原因: 方向污染 — 想按K(低段)结果按了6K(中段)
  解决:
    K 派生 = 松开所有方向 + K（中立+K）
    6K 派生 = 按→ + K
    4K 派生 = 按← + K
    关键: 先松开方向回中立，再选派生

问题4: 迅雷脚取消绿冲连不上
  原因: OD迅雷脚(236KK)取消DR的时机
  解决: 236KK 命中后 → 立刻 66 或 MP+MK → DR出来后接连招
```

### 2.3 迅雷脚在 Hitbox 上的输入

```
基础迅雷脚:
左手: ↓ → ↓+→ → →      (就是 236 波动拳指令)
右手: 最后的→同时按 K     (任意脚键)

低段派生 (Kazekama / K):
迅雷脚出来后 → 松开所有方向 → 按 K
Hitbox: 左手全松 → 右手按 K
关键: 确保左手完全松开，不要残留方向输入

中段派生 (Gorai / 6K):
迅雷脚出来后 → 按 → + K
Hitbox: 左手按→ → 右手按 K

旋踢派生 (Senka / 4K):
迅雷脚出来后 → 按 ← + K
Hitbox: 左手按← → 右手按 K

OD迅雷脚:
左手: ↓ → ↓+→ → →
右手: → 同时按 两个脚键(KK)
消耗 2 格 Drive
```

### 2.4 迅雷脚连招专练

#### 基础：迅雷脚确认
```
练习1: 2MK xx 236LK（蹲中脚取消轻迅雷脚）
  → 这是Ken最基础的确认
  → 先不接派生，只练 2MK xx 236LK 出来
  → 目标: 10/10 左右两边

练习2: 2MK xx 236LK > K（接低段派生）
  → 迅雷脚命中后接低段
  → 练时机: 236LK出来后等一拍再按K
  → 目标: 10/10
```

#### 中级：迅雷脚接 DR
```
练习3: 2MK xx 236LK xx DR > 2MP > 5HP xx 623HP
  分解:
  (A) 2MK xx 236LK — 蹲中脚取消迅雷脚 (先练好)
  (B) 236LK xx DR — 迅雷脚取消绿冲
      方法: 236LK 后立刻 MP+MK 或 66
      注意: 236LK 必须命中才能取消！
  (C) DR > 2MP > 5HP — 绿冲后打两下
  (D) 5HP xx 623HP — 重拳取消升龙

  全连: (A) > (B) > (C) > (D)
  目标: 5/10 先能出来 → 逐步提高到 8/10
```

#### 高级：迅雷脚中段二择
```
练习4: 236MK > 6K（中段派生 overhead）
  → 迅雷脚中段打防蹲的对手
  → 练习在 236MK 出来后精确输入 6K
  → 先不管命中后连招，只练派生选择

练习5: 迅雷脚二择
  → dummy 设 Random Block
  → 你用 236MK:
    → 命中 → 接 K(低段) > 追击
    → 被防 → 下次用 6K(中段) / 急停投
  → 目标: 形成"看到命中选什么，被防选什么"的意识
```

#### 进阶：OD 迅雷脚路线
```
练习6: 5MP > 5HP xx 236KK(OD迅雷脚) xx DR > 5MP > 5HP xx 623HP
  → OD迅雷脚可以取消DR！
  → 消耗: 2格(OD) + 3格(DR) = 5格 Drive
  → 伤害极高，Ken 的顶级连招

  分解:
  (A) 5MP > 5HP xx 236KK — 确认接OD迅雷
  (B) 236KK xx DR(66或MP+MK) — OD迅雷取消绿冲
  (C) DR > 5MP > 5HP xx 623HP — 绿冲后续
```

### 2.5 迅雷脚从易到难练习路线

## 六、关键心态提醒

```
1. 绿冲练不好是正常的 — 这是 SF6 最难的执行之一
   → 职业选手也会掉绿冲
   → 先用 MP+MK 替代输入法，降低难度

2. 迅雷脚不熟就少用，不要强行多择
   → 先只用 2MK xx 236LK > K（安全低段）
   → 熟练后再加 6K(中段) 和 急停投

3. 实战中"能出来"比"最优路线"重要100倍
   → 训练模式7/10 ≈ 实战3/10（压力下成功率减半）
   → 所以训练要追求 9/10 才够实战用

4. 不要同时练太多东西
   → 这周只专注绿冲 → 下周加迅雷脚 → 再下周加电刃/起攻
   → 一次改善一个方面
```
