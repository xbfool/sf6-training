# 第四章：角色篇 — 速攻型 (Rushdown)

---

## 1. Cammy (嘉米)

### 角色定位
最纯粹的速攻角色之一。极快的前冲速度、优秀的帧数据、**Cannon Strike** 改变跳跃轨迹、**Hooligan Combination** 提供丰富二择。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Spiral Arrow 螺旋箭 | 236K | 低姿态突进，LK版安全 |
| Cannon Spike 加农炮升踢 | 623K | 无敌对空 |
| Cannon Strike | (前跳) 214K | 空中下压踢，改变轨迹 |
| Hooligan Combination | 236P | 翻滚接近+派生 |
| → Razor Edge Slicer | 236P > 不按 | 低段滑踢 |
| → Cannon Strike | 236P > K | 空中踢 |
| → 空中投 | 236P > LP+LK | 抓取 |
| Quick Spin Knuckle | 214P | 前进打击，穿飞道具 |
| SA1 Spin Drive Smasher | 236236K | 突进超杀 |
| SA3 Delta Red Assault | 236236P | Level 3 |

### 推荐 BnB 连招

```
基础:
5MP > 2MP xx 236LK                    ~1800 基础确认
2MK xx 236LK                          ~1400 核心牵制确认
2LP > 5LP > 2MK xx 236LK              轻攻击确认
5HP xx 623HK                          简单惩罚

Drive Rush:
2MK xx 236LK xx DR > 5MP > 2HP xx 623HK    ~3000 主力BnB
5HP xx 236MK xx DR > 5MP > 2HP xx 623HK    重拳DR路线

跳入:
j.HP > 5HP > 2HP xx 623HK             ~2600
(空中)214MK(深中) > 2LP > 5MP > 2HP xx 623HK  Cannon Strike确认

角落:
(角落) 5HP > 2HP xx 236HK > 5MP > 623HK  螺旋箭追击
```

### Cannon Strike 技巧
- 深中的 Cannon Strike (214K) 给予大有利帧
- 改变跳跃轨迹，让对手对空困难
- 练习不同高度的 Cannon Strike：越低越有利

### Hooligan 三择
```
236P(翻滚) → K(Cannon Strike中段)
236P(翻滚) → 不按(Razor Edge低段)
236P(翻滚) → LP+LK(空中投)
```

### 起攻套路
- **623HK 后**: 前步 > meaty 5MP / 投
- **Cannon Strike 深中**: 落地即可压制
- **Hooligan**: 击倒后 236P 翻滚 > 三择
- **2HK 后**: 前步 meaty 或 Hooligan

### 优势/劣势
- (+) 前冲极快，施压顶尖；2MK 范围大；连招简洁
- (-) 没有飞道具；体力偏低；被压时脱身手段有限

## 2. Kimberly (金伯利)

### 角色定位
忍者速攻型，极高机动性。**疾跑 (Sprint)** 派生系统 + **喷漆炸弹** 设置 + 空中移动。二择能力顶级。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| 疾跑 Sprint | 236K | 快速前冲+派生 |
| → Emergency Stop | 236K > K | 急停 |
| → Shadow Slide | 236K > LK | 低段滑踢 |
| → Arc Step | 236K > MK | 空中踩踏 |
| → Elbow Drop | 236K > HK | 中段突进 |
| 喷漆炸弹 Spray Can | 214P | 设置喷漆罐 |
| Nue Sashi | 236P | 突进打击 |
| Genius at Play | 623P | 对空 |
| 空中旋转脚 | (空中) 214K | 空中攻击 |
| SA1 Bushin Beats | 236236K | 连续打击 |
| SA3 Ninjastar Cypher | 236236P | Level 3 |

### 推荐 BnB 连招

```
基础:
2MP > 5MP xx 236LK > LK(Shadow Slide)     基础确认
2MK xx 236LK > LK                         下中脚确认

Drive Rush:
5MP > 5HP xx DR > 5MP > 5HP xx 236MK > MK(Arc Step) > j.MK    DR空中追击
2MK xx 236LK xx DR > 2MP > 5HP xx 623HP                        下段DR

跳入:
j.HP > 5HP > 2HP xx 236MK > MK > j.MK    空中追击
j.HP > 5MP > 5HP xx 236K > HK             接肘打
```

### 疾跑三择（核心混淆）
```
击倒后 236K(疾跑):
→ Emergency Stop > 投           急停投
→ Shadow Slide(低段)            下段突进
→ Arc Step(踩头中段)            中段跳跃
→ Elbow Drop                   中段突进
```
对手需要猜测 4 种选项，猜错吃连招。

### 喷漆炸弹应用
- 击倒后设置 214P → 对手起身面对炸弹+你的攻击双重威胁
- 中立时设置 → 控制空间

### 优势/劣势
- (+) 速度极快；二择能力顶级；空中机动高
- (-) 体力偏低；被看穿惩罚大；远距离缺乏飞道具；对空一般

## 3. Rashid (拉希德)

### 角色定位
机动力最强角色之一。**旋风 (Whirlwind Shot)** 飞道具创造掩护。**Arabian Cyclone** 翻越对手制造左右择。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Whirlwind Shot 旋风弹 | 236P | 缓慢前进的旋风柱 |
| Spinning Mixer | 236K | 多段旋转攻击 |
| Eagle Spike | 214K | 突进踢 |
| Arabian Cyclone | 214P | 前翻+派生 |
| Wing Stroke | (空中) 214P | 空中突进 |
| SA1 Super Spinning Mixer | 236236K | 超旋转 |
| SA2 Ysaar | 236236P | 大旋风 |
| SA3 Altair | 214214P | Level 3 |

### 推荐 BnB 连招

```
基础:
5MK > 5MP xx 236LK(Spinning Mixer)    核心确认 (LP Mixer安全留对手站立)
2MK xx 214LK(Eagle Spike)             下段确认
5HP xx 236P(旋风设置)                  重拳取消旋风

Drive Rush:
5MK > 5MP xx DR > 5MP > 5HP xx 236MK   DR扩展
2MK xx 214LK xx DR > 5MP > 5HP xx 236MK  Eagle Spike DR取消
5HP xx 236P xx DR > 5MP > 5HP xx 214HK   旋风DR连招

跳入:
j.HP > 5HP > 2HP xx 236HK              接重旋转
j.HP > 5MP > 5HP xx 236P xx DR > 5MP > 5HP xx 214HK  DR最大化
```

### 旋风设置战术

```
击倒后 236P(旋风):
→ 旋风缓慢前进，在对手起身时到达
→ 对手必须防御旋风，你安全接近
→ 旋风掩护下 投/打 二择
→ OD旋风(236PP) 更大更持久，压制更强
```

### Arabian Cyclone 左右择
```
击倒后 214P(翻越对手):
→ 出现在对手身后
→ 对手防守方向混乱
→ 配合旋风效果翻倍
```

### 优势/劣势
- (+) 机动性极强；旋风设置独特；左右择强；Arabian Cyclone翻越
- (-) 伤害偏低；需要旋风配合；被封住机动性优势大减；对空一般

## 4. Terry (泰瑞)

### 角色定位
DLC 角色（饿狼传说系列）。全面进攻型，保留饿狼传说操作特色。**低体力时超杀强化**是独特翻盘机制。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Power Wave 能量波 | 236P | 地面飞道具 |
| Burn Knuckle 燃烧拳 | 214P | 突进拳 |
| Rising Tackle 上升螺旋 | 623K | 对空上升旋转 |
| Crack Shoot 裂射踢 | 214K | 前翻踢，过飞道具 |
| Power Dunk 能量灌篮 | 623P | 突进上升打击 |
| Power Charge | 236K | 前冲蓄力攻击 |
| SA1 Power Geyser | 236236P | 经典地面喷发 |
| SA2 Buster Wolf | 236236K | 突进超杀 |
| SA3 | 214214P | Level 3（低体力时强化） |

### 推荐 BnB 连招

```
基础:
5LP > 5MP > 5HP(Target Combo) xx 214MP    TC接燃烧拳
2MK xx 214LP                               下段确认
5HP xx 623MP(Power Dunk)                   重拳接灌篮

Drive Rush:
5MP > 5MK xx DR > 5MP > 5HP xx 623MP       DR接Power Dunk
2MK xx 214LP xx DR > 5MP > 5HP xx 623HK    下段DR路线

跳入:
j.HK > 5HP > 2HP xx 623HK                  接重上升螺旋
j.HK > 5HP > 2HP xx 214HP > 623HP          高伤害路线

超杀:
确认 xx 623HP(Power Dunk) xx SA1(Power Geyser)  灌篮接超杀
确认 xx 214HP xx SA2(Buster Wolf)                燃烧拳接Buster Wolf
```

### 特殊系统
- **Target Combo**: 5LP>5MP>5HP 三连锁，Terry 特有
- **低体力强化超杀**: 体力低于 ~25% 时，SA 有额外效果和伤害
- **Crack Shoot 越头**: 214K 可以翻越对手+飞道具

### Power Dunk 的重要性
- 近身 oki 极强（命中后紧贴对手）
- 对手不后滚时可以直接 meaty
- 连招终结的首选

### 优势/劣势
- (+) 工具齐全；低体力翻盘能力；Target Combo灵活；Crack Shoot越弹
- (-) 操作体系独特需适应；部分招式空了恢复慢；中距离有空档
