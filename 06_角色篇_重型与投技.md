# 第六章：角色篇 — 重型与投技 (Grapplers & Heavies)

## 1. Zangief (桑吉尔夫)

### 角色定位
纯投技角色，全游戏最高单次投技伤害+最高体力(10500)。核心：缩短距离 > SPD投。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| SPD 螺旋打桩机 | 360+P | 核心指令投，LP远/HP高伤 |
| Siberian Express | 360+K | 跑投，前冲后抓取 |
| Double Lariat | 214P | 旋转攻击，上半身打击无敌 |
| Borscht Dynamite | 632146P | 近距离高伤害指令投 |
| Tundra Storm | 63214K | 对空投技 |
| SA1 | 236236K | 前冲投超杀 |
| SA2 Cyclone Lariat | 214214P | 旋风攻击，有无敌 |
| SA3 Bolshoi Storm | 720+P | 720超投，巨额伤害 |

### Hitbox 360/720 输入（关键！）

**360 推荐输入（从蹲防开始，避免跳跃）：**
```
已经蹲防(↓+←) → 松开↓ → 按→ → 按↑+P → 完成
即: 1(下后) > 4(后) > 6(前) > 8+P(上+拳)
```

**Tick Throw SPD（实战最常用）：**
```
2LP(出招时按着↓) → 恢复帧中: 6 > 8 > 4 + P
= 轻拳后缓冲SPD
```

**Drive Rush SPD:**
```
2MK xx DR(66) → 冲刺中: ↓ > ← > ↑ + P
利用DR前冲动画隐藏360输入
```

**720（SA3）冲刺缓冲法：**
```
66(前冲) → 冲刺中: ↓←↑→↓←↑+P
前冲动画掩盖720，避免跳起
```

### 推荐 BnB 连招

```
基础:
5LP > 5LK > 360+LP(SPD)               tick throw投
2MP > 5MP xx 214P(Lariat)              中拳接旋风
2LP > 2LP > 2LK xx 236236K(SA1)       轻确认接超杀

Drive Rush:
2MK xx DR > 5HP > 6HP xx 632146P(Borscht)  ~3000 DR大伤害
2MK xx DR > 2MP > 5HP xx 360+HP(HP SPD)    DR接SPD

惩罚:
5HP > 6HP xx 214214P(SA2) > 720+P(SA3)     最大伤害(满槽)
j.HP > 5HP > 6HP xx 632146P                 跳入大伤害

DI:
DI(墙贴) > 5HP > 6HP xx OD 360+P           DI惩罚
```

### 投/打 二择（Zangief 灵魂）
```
接近对手后:
选项A: SPD(投) — 对手防御就吃投
选项B: 5MP/2MP(打击) — 对手跳就被打中
选项C: Shimmy(后退骗投) — 对手拆投就打空，你惩罚

对手必须猜:
  防 → 被投
  跳 → 被打/被对空
  拆投 → 被shimmy惩罚
  按键 → 被帧陷阱打Counter
```

### 优势/劣势
- (+) 单次投技伤害全游戏最高；体力最高；投/打二择恐怖；DR接近手段
- (-) 机动性差移动慢；对远程角色极其挣扎；没有飞道具；必须接近才能打

## 2. Marisa (玛丽莎)

### 角色定位
重型打击手。单次打击伤害全游戏最高，多招式自带 **Armor（霸体）**，可蓄力招式破防。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Gladius | 236P (可蓄力) | 前冲拳，蓄力版有armor |
| Phalanx | 214P | 反击/对空拳，有armor |
| Dimachaerus | 236K | 前踢突进 |
| Scutum 盾牌架势 | 214K | 吸收攻击+派生 |
| → Scutum > P | 打击追加 |
| → Scutum > K | 前冲追加 |
| Tonitrus | 623P | 升龙式攻击，OD无敌 |
| SA2 Megalomania | 214214P | 有armor超杀 |
| SA3 Goddess of Hunt | 236236K | Level 3 |

### 推荐 BnB 连招

```
基础:
2LK > 2LP xx 236LP(Gladius)             轻确认
2MP > 5MP xx 236LP                       中拳确认
5HP xx 236MK(Dimachaerus) > 236LP        重拳连招

Drive Rush:
2MK xx DR > 5HP xx 236MK > 236HP         ~3200 DR高伤害
5HP xx 214MP(Phalanx) > 236MK             Phalanx追击

惩罚/PC:
(PC) 5HK > 5HP xx 236MK > 236HP          PC大伤害
DI(墙贴) > 5HP xx 214MP > 236MK > SA3    DI最大伤害
j.HP > 5HP xx 236MK > 5MP xx 236MK > SA1  跳入超杀
```

### Armor 战术
```
蓄力Gladius 236[HP]:
→ 对手按键被armor吸收 → 蓄力Gladius命中
→ 相当于"霸体突进"
→ 不要滥用：被投技/多段攻击/看穿会被惩罚

Scutum 214K:
→ 吸收一次攻击后选择:
  > P: 反击打击
  > K: 前冲接近（可接投或打击）
→ 相当于读心防御后的反击
```

### 起攻: 蓄力Gladius meaty
```
击倒后(角落):
→ 236[HP](蓄力Gladius) meaty
→ 对手起身时被armor版打击压制
→ 对手DP会被armor吸收后反击
→ 极其强力的角落起攻
```

### 优势/劣势
- (+) 单次伤害极高；armor招式穿透攻击；蓄力给巨大压力；血量高
- (-) 招式帧数慢；机动性一般；armor被投/多段破解；被拉远接近难

## 3. Manon (玛侬)

### 角色定位
柔道+芭蕾投技角色。**Medal 等级系统** — 投中次数越多越强，且**跨回合/跨局保持**！

### Medal System（核心机制！）
```
每次指令投 (Manege Dore / 360+K) 命中 → Medal +1
每次 SA1 命中 → Medal +1

Medal 等级:
Lv1: 基础伤害
Lv2: 伤害+~20%
Lv3: 伤害+~40%
Lv4: 伤害+~60%
Lv5: 伤害+~80-100% (恐怖！)

关键: Medal 跨回合/跨局不重置！
= 比赛越打投技越痛
```

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Manege Dore | 360+K | 核心指令投，命中Medal+1 |
| Renverse | 623K | 对空投，命中Medal+1 |
| Rond-point | 236K | 旋转踢，连招终结 |
| Demi-Grand-Rond | 214K | 后旋踢，回避效果 |
| En Haut | 623P | 上升攻击，OD无敌 |
| SA1 Arabesque | 236236K | 前冲投超杀，命中Medal+1！ |
| SA3 Pas de Deux | 236236P | Level 3 |

### Hitbox 360 输入

**与 Zangief 相同方法，但按 K（脚键）：**
```
蹲防(↓+←) → 松开↓ → 6 > 8 > 4 + K
Tick: 2LK > 恢复帧中 6 > 8 > 4 + K
DR: 2MK xx DR > 冲刺中 ↓ > ← > ↑ + K
```

**SA1 (236236K) 比 360+K 更容易输入**
→ 在不确定时用 SA1 替代指令投（同样升 Medal！）

### 推荐 BnB 连招

```
Medal 优先路线（核心策略）:
2LK > 2LP xx 236236K(SA1)              轻确认接SA1 升Medal！
5LP > 5LK > 360+LK(Manege Dore)        tick throw 升Medal！
2MK xx DR > 5MP > 6MP xx 236236K(SA1)   DR确认接SA1 升Medal！

伤害路线:
5HP > 6MP xx 236HK(Rond-point)          惩罚连招
2MK xx DR > 5MP > 6MP xx 236HK          DR标准
j.HP > 5HP > 6MP xx DR > 5HP > 6MP xx 236HK  跳入DR延伸

最大伤害:
DI(墙贴) > 5HP > 6MP xx 236236K(SA1)    DI接SA1 升Medal
(PC) 5HK > 5HP > 6MP xx 236236P(SA3)    PC最大伤害
```

### Medal 策略思路
```
回合1-2: 优先用SA1/360+K 升Medal (Lv1→Lv2→Lv3)
中期: Medal Lv3+ 后投技已经很痛，开始施压
后期: Medal Lv5 时一次投技几乎半血

对手心理: "她Medal已经很高了，被投一次就半血..."
→ 对手极度紧张 → 更容易犯错 → 你打击确认也更容易
```

### 优势/劣势
- (+) Medal系统越打越强；跨回合保持；投/打二择伤害高；SA1升Medal又能确认
- (-) 前期投技伤害偏低；对方跳避投需要对空；没有飞道具；对zoner接近困难

## 4. Honda (本田)

### 角色定位
相扑蓄力角色。头槌突进力极强，百裂张手压制，Hitbox 上蓄力角色优势明显。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Headbutt 头突 | [4]6P | 蓄力前冲头槌，核心突进 |
| Sumo Smash 百貫落 | [2]8K | 上升下砸，对空/改轨迹 |
| 百裂張手 HHS | 236P | 连续掌击，压制 |
| Oicho 指令投 | [4]6K | 蓄力指令投 |
| Sumo Spirit | 214K | armor前冲+派生 |
| SA1 | [4]6[4]6P | 超级头槌 |
| SA2 | [2]8[2]8K | 多段踩踏，有无敌 |
| SA3 | 236236K | Level 3 |

### Hitbox 蓄力技巧（Honda 专用）

```
基础蓄力:
按住 ←(4) 至少40帧 → 松开+按 →(6)+P = 头槌

边走边蓄力:
按住 ← 时出 5MP/5HP（不需要方向输入的普通技）
→ 一边蓄力一边牵制

SOCD 蓄力切换:
按住 ← → 快速按 → 释放 → 松开 → = 中立 → 只剩 ←
→ 利用 SOCD 快速在蓄力和中立间切换

Flash Kick 式 Sumo Smash:
按住 ↓+← → 右手拇指按 ↑+K
→ SOCD: ↑+↓ = 中立，但过渡帧被游戏读取为有效的 [2]8
```

### 推荐 BnB 连招

```
基础:
2LP > 2LP > 5LP xx [4]6LP(Headbutt)    轻攻击蓄力确认
5MP > 5MP xx 236MP(HHS)                中拳接百裂
2MK xx [4]6MP(Headbutt)                蹲中脚接头槌

Drive Rush:
2MK xx DR > 5HP xx 236HP(HHS) > [4]6HP  DR百裂接头槌
DI(hit) > 5HP xx OD236P(OD HHS) > [4]6HP  DI高伤害

跳入:
j.HP > 5HP xx 236HP(HHS) > [4]6HP      跳入百裂头槌

蓄力超杀:
2MP > 5HP xx DR > 5HP xx 236HP > [4]6HP xx SA3  最大伤害
```

### 头槌战术
```
LP Headbutt [4]6LP: -2帧，基本安全，用于确认和突进
MP Headbutt [4]6MP: 更远更痛，但-4可被惩罚
HP Headbutt [4]6HP: 最远最痛，但不安全
OD Headbutt [4]6PP: 快速安全，突进利器

实战: 大多数情况用 LP 头槌安全确认
惩罚时用 HP 头槌追求伤害
```

### 起攻
- **HK Sumo Smash 后**: 帧数非常有利，可直接 meaty
- **角落 Oicho 后**: 紧贴对手，2LP meaty / 投 二择
- **OD 百裂 (+4)**: 命中后直接投/打二择

### 优势/劣势
- (+) 头槌突进强穿飞道具；近身伤害高；百裂压制力强；适合新手
- (-) 蓄力限制灵活性；招式偏直线；高水平头槌可被反应惩罚
