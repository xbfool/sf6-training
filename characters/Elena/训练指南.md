# Elena (艾莲娜) 专属训练指南

## 一、角色定位与核心打法

Elena 是**卡波耶拉 (Capoeira) 踢技角色**，来自 SF3 系列。核心打法围绕以下三根支柱：

1. **SA2 Healing (214214K) 回血**：SF6 中唯一可以回复体力的角色！这个能力从根本上改变了比赛的资源管理和节奏。每一格超杀槽都面临"打伤害还是回血"的战略决策。
2. **中距离踢技控制**：Spin Scythe (236K) 旋转低踢、Rhino Horn (6HK) 前冲踢、各种独特的踢技轨迹——Elena 的腿长，中距离是她的主场。部分踢技有回避属性，可以闪过对手的攻击。
3. **Scratch Wheel (623K) 升龙式对空**：OD 版有无敌帧，既是对空工具也是防守反转选项。

**Elena 的节奏**：中距离踢技打点 → 确认连招 → 击倒对手 → 安全时用 SA2 回血 → 血量差距越拉越大 → 对手被迫冒险 → 惩罚对手的冒险。

**核心战略决策（贯穿全局）**：
```
超杀槽满 → 三个选择：
1. SA1（伤害）→ 连招终结，稳定输出
2. SA2 Healing（回血）→ 续命/拉大血量差距
3. SA3（最大伤害）→ 击杀/翻盘
```
没有"正确答案"，需要根据场面判断。这就是 Elena 的战略深度。

**新手友好但上限高**：Elena 的基础连招不难，但资源管理决策需要大量实战经验。低段位可以简单打 BnB + 回血；高段位需要精准判断每格槽的最优用法。

## 二、核心连招

### BnB 连招（必练）

这些是日常对战中使用频率最高的连招，必须练到100%成功率。

```
1. 2LK > 2LK xx 236LK（Spin Scythe）
   用途：轻确认连招，近距离防守反击或小技确认
   伤害：约1100
   要点：两个 2LK 确认命中后取消旋转低踢，安全可靠

2. 5MK > 2MK xx 623MK（Scratch Wheel）
   用途：中距离标准确认，踢技接升龙式收尾
   伤害：约1800
   要点：5MK 命中确认后接 2MK 取消 Scratch Wheel，升龙收尾伤害高

3. 2MK xx 236MK
   用途：蹲中脚确认，Elena 的 footsies 主力工具
   伤害：约1300
   要点：2MK 距离远，确认命中后取消 Spin Scythe
```

### Drive Rush 连招

消耗 3 格 Drive Gauge，用于将确认命中转化为大伤害。

```
1. 5MK > 6MK xx DR > 5HP xx 236HK
   用途：DR 延伸主力连招
   伤害：约2600
   要点：6MK 取消 DR 后接 5HP 打出高伤害，236HK 收尾

2. 2MK xx DR > 5MP > 5HP xx 236HK > SA1
   用途：DR 超杀收尾最大伤害
   伤害：约3500+
   要点：蹲中脚确认后 DR 延伸并用 SA1 收尾，当需要伤害击杀时使用
```

### 惩罚连招

对手大失误时使用的最大伤害连招。

```
1. 5HP > 5HK xx 623HK（Scratch Wheel）
   用途：标准重惩罚
   伤害：约2400
   要点：升龙被防等大负帧机会，5HP 起手打出最大伤害

2. 5HP > 5HK xx 236HK > SA1
   用途：超杀惩罚收尾
   伤害：约3500+
   要点：当判断需要用伤害击杀而非回血时

3. 5HP > 5HK xx 236HK > SA3
   用途：Level 3 最大伤害惩罚
   伤害：约4500+
   要点：确定击杀用，血量足够不需要回血时
```

### 跳入连招

跳入命中后的追击。

```
1. j.HK > 5HK xx 623HK
   用途：基础跳入连招
   伤害：约2600
   要点：j.HK 深打确保 5HK 连上，623HK 收尾

2. j.HK > 5HP > 5HK xx 236HK > SA1/SA3
   用途：跳入最大伤害
   伤害：约3800+
   要点：跳入机会难得，尽量打出最大伤害
```

### 超杀收尾

```
1. ... xx 236HK > SA1
   用途：连招尾部 Spin Scythe 取消 SA1，稳定伤害收尾
   决策点：使用 SA1 意味着放弃了回血机会

2. ... xx 623HK > SA1
   用途：Scratch Wheel 取消 SA1
   决策点：同上

3. SA3
   用途：Level 3 最大伤害，击杀用
   决策点：使用 SA3 意味着消耗全部槽，无法回血

4. SA2 Healing (214214K)
   用途：不是连招收尾！而是击倒对手后安全使用
   时机：对手倒地期间、全屏距离、对手 Burnout 时
```

## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心原则：中距离踢技保持距离+回血拖时间，利用持久战优势。**

具体策略：
- **距离控制是关键**：保持在 2MK/5MK 尖端距离，这是投技角色 SPD 够不到的距离。Elena 的腿长，中距离踢技范围好，天然克制需要近身的投技角色。
- **Spin Scythe (236K) 低踢牵制**：旋转低踢可以在中距离打点，命中倒地后安全回血 (SA2)。
- **Rhino Horn (6HK) 前冲踢**：中距离读心对手前进时用 6HK 突进打击，命中后有利。
- **回血策略是核心优势**：对投技角色的持久战中，Elena 可以用 SA2 不断回血。投技角色一旦投不到你就没有好的远距离手段，你可以安全回血拉大差距。
- **被近身时**：OD Scratch Wheel (OD 623K) 有无敌，可以反转。Drive Reversal 推开。后跳逃脱投技范围。
- **不要做的事**：不要在角落和投技角色对峙。用中距离踢技和回血能力把比赛节奏拉到你的主场。

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心原则：稳守反击，回血弥补防守失误的血量损失。**

具体策略：
- **对空第一优先**：Scratch Wheel (623K) 对空，OD 版有无敌。速攻角色频繁跳入，对空成功后前步压制。
- **蹲防为主**：速攻角色压制强，蹲防等间隙。被连续压制时用 Drive Reversal 脱身。
- **回血弥补失误**：这是 Elena 对速攻角色的独特优势。被连招打了一套后击倒对手回血 (SA2)，等于"免费"挨了一套。速攻角色最怕遇到打不死的对手。
- **中距离踢技反击**：速攻角色前冲间隙用 2MK xx 236MK 打点。5MK 的踢技轨迹有回避属性，可能闪过对手的某些攻击。
- **Scratch Wheel 防守反转**：OD 623K 有无敌帧，被压制时可以读心反转。但被防会被大惩罚，不要滥用。
- **Perfect Parry 对手套路**：速攻角色有固定的压制套路（如 Cammy 的 2MK xx Spiral Arrow），练习 Perfect Parry 这些常见套路。

### vs 远程角色 (JP/Guile/Dhalsim)

**核心原则：耐心接近，接近后打点+回血，用持久战拖垮远程角色。**

具体策略：
- **接近路线**：
  - 前走 + Drive Parry 弹反飞道具逐步接近
  - Drive Rush 穿过飞道具间隔
  - Rhino Horn (6HK) 中距离前冲踢突进
  - DI 吸收飞道具继续前进（偶尔使用）
- **不要急**：Elena 有回血能力，被飞道具打到也不怕——因为之后可以回血。远程角色最怕遇到打不死、还能回血的对手。
- **接近后打点+击倒→回血**：进入中距离后用踢技打点，击倒对手后用 SA2 回血。这样即使接近过程中挨了些飞道具伤害，回血后血量差距依然对你有利。
- **Drive Gauge 管理**：接近过程中 Drive Parry 和 DR 会消耗 Drive Gauge。注意不要在接近过程中就 Burnout。
- **对 Guile 特别注意**：不要中距离跳——Flash Kick 对空极痛。地面接近为主。进入中距离后 Guile 的牵制技不如 Elena 的踢技灵活。
- **对 Dhalsim 注意**：Dhalsim 的伸缩手脚恢复帧长，用 Rhino Horn 惩罚他的空振。

### vs 重型角色 (Marisa/Honda)

**核心原则：不硬碰硬，中距离踢技打点+回血消耗战。**

具体策略：
- **中距离是 Elena 的主场**：重型角色近身伤害极高，但中距离够不到 Elena 的踢技。用 2MK、5MK、Spin Scythe 在安全距离打点。
- **回血消耗战**：重型角色单次伤害高但命中机会少。Elena 每次被打后都可以回血弥补，长期来看重型角色会"打不死"Elena。
- **破 Armor**：OD Spin Scythe 多段攻击可以破 Marisa 的 armor。投技也可以破 armor。
- **Honda 头槌**：LP 版 (-2) 基本安全，HP 版大负帧可以用 5HP > 最大伤害惩罚。训练模式练习区分。
- **Rhino Horn 距离控制**：6HK 前冲踢可以在重型角色想要前进时打断他们。
- **避免角落**：和重型角色在角落打是最糟糕的选择。用 Drive Reversal 或 OD Scratch Wheel 脱身。

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心原则：踢技对抗 footsies+回血弥补波动拳伤害+精确惩罚升龙拳。**

具体策略：
- **波动拳对策**：
  - 全屏：Drive Parry 弹反恢复 Drive Gauge，或前走接近
  - 中距离：预判波动拳用 Rhino Horn (6HK) 穿过
  - 近距离：不要跳！升龙对空极痛。用 Drive Parry 或 OD Scratch Wheel
- **升龙拳惩罚（必练！）**：
  - 防住升龙 → 5HP > 5HK xx 623HK（约2400伤害）
  - 有 SA：5HP > 5HK xx 236HK > SA1（约3500伤害）
  - 决策点：惩罚用 SA1 打伤害还是省下来回血？要根据血量差距和比赛局势判断
- **中距离 footsies**：Elena 的 2MK 和 5MK 距离与波动拳系角色的 footsies 距离重叠。Elena 的踢技有独特轨迹和回避属性，可能在交换中占优。
- **回血策略**：被波动拳 chip 或小连招打到后，击倒对手回血。波动拳系角色会发现"每次打的伤害都被回复了"。
- **SA 决策核心场景**：
  - 你血多对手血少 → SA2 回血拉大差距，稳赢
  - 你血少但够一套连招击杀 → SA1/SA3 打伤害
  - 你血少对手也血少 → 看谁先犯错，保留 SA 灵活应对

## 四、训练模式练习法

### 练习1：对空反应训练

**设置方法**：
1. 训练模式 > Dummy 设置 > 随机跳入
2. Dummy 角色选 Cammy（跳弧快，练好了其他都能对空）

**练习内容**：
- 看到对手跳 → 623K（Scratch Wheel）对空
- 中远距离跳：623MK 或 5HK 对空
- 近距离跳：OD 623K（有无敌帧）
- 对空成功后：前步 meaty 或安全距离回血 (SA2)

**目标**：连续 10 次对空成功率 > 80%。

**每日时长**：5 分钟。

### 练习2：确认连招训练

**设置方法**：
1. Dummy 设置 > Guard 设为 Random
2. 你打出 2MK、5MK、2LK

**练习内容**：
- 打出 2MK：
  - 命中 → xx 236MK（Spin Scythe）
  - 被防 → 停手或只出轻版 Spin Scythe
- 打出 5MK：
  - 命中 → 2MK xx 623MK（Scratch Wheel）
  - 被防 → 停手
- 打出 2LK：
  - 命中 → 2LK xx 236LK
  - 被防 → 停手或继续轻攻击压制

**目标**：命中/被防判断准确率 > 90%，连招成功率 > 95%。

**每日时长**：5 分钟。

### 练习3：惩罚训练

**设置方法**：
1. Dummy 设置 > 录制以下动作：
   - 录制1：623HP（重升龙拳）
   - 录制2：HP+HK（DI）
   - 录制3：重必杀技（如 Honda HP 头槌）
2. 设为随机回放

**练习内容**：
- 防住升龙拳 → 5HP > 5HK xx 623HK
- 有 SA 槽 → 5HP > 5HK xx 236HK > SA1
- 防住 DI → 反 DI 或 5HP > 连招
- 防住重必杀 → 快速惩罚

**附加决策练习**：惩罚时练习快速判断——用 SA1 打伤害还是省槽回血？
- 对手血少可击杀 → SA1 收尾
- 自己血少需要续命 → 省槽，之后回血
- 对手血多打不死 → 省槽，控制节奏

**目标**：惩罚连招 + SA 决策在 1 秒内完成。

**每日时长**：5 分钟。

### 练习4：起攻/压制训练

**设置方法**：
1. Dummy 设置 > 起身后行动 Random
2. 你先用连招击倒 Dummy

**练习内容**：

**击倒后选项训练**：
1. 连招击倒对手
2. 判断距离和血量后选择：
   - **选项A（需要伤害）**：前步 meaty 5MK → 命中确认接连招
   - **选项B（需要回血）**：后退安全距离 → SA2 Healing 回血
   - **选项C（混合）**：前步 meaty → 命中后继续压制，被防就退回安全距离回血

**SA2 回血时机练习**：
1. 击倒对手后立即 214214K 回血
2. 计时：对手起身前是否能完成回血动作
3. 练习判断不同击倒后的回血时间窗口

**压制串练习**：
- 5MK(+帧) > 2MK xx 236MK → 踢技压制串
- 5MK > 微延迟 > 5MK → 帧陷阱
- 前步投 / 前步 2MK → 投打二择

**目标**：击倒后 100% 做出选择（压制或回血），不要放空。

**每日时长**：10 分钟。

### 练习5：角色专属技能训练 — SA2 回血决策

**设置方法**：
1. 训练模式 > 设置自己和 Dummy 的初始血量为不同数值
2. 设置超杀槽等级

**练习内容**：

**A. 回血时机判断训练**：
设置不同场景，练习判断是否回血：
- 场景1：你 70% 血，对手 50% 血，1 格槽 → 回血（拉大差距）
- 场景2：你 30% 血，对手 20% 血，2 格槽 → SA1 击杀（一套带走）
- 场景3：你 30% 血，对手 80% 血，3 格槽 → SA2 回血（续命后重新比赛）
- 场景4：你 50% 血，对手 10% 血，1 格槽 → SA1 收尾（击杀优先）

**B. 安全回血窗口练习**：
- 击倒后回血：不同击倒招式后的回血时间窗口
- 全屏回血：对手被推到全屏时的回血时机
- Burnout 后回血：对手 Burnout 时安全回血（对手无法快速接近）

**C. Scratch Wheel 对空 + 回血循环**：
1. 对空成功 (623K) → 对手倒地
2. 安全距离 SA2 回血
3. 对手起身 → 回到中距离踢技控制
4. 形成"对空→回血→控制→对空"的循环

**D. Rhino Horn (6HK) 距离感训练**：
- 不同距离使用 6HK 前冲踢
- 练习 6HK 命中确认后的追击选项
- 注意被防时的帧数——被防后是否安全

**E. Spin Scythe (236K) 版本区分**：
- 236LK：短距离，安全
- 236MK：中距离，连招主力
- 236HK：长距离，伤害高但恢复帧长
- 练习不同距离选择不同版本

**目标**：SA2 回血决策在 0.5 秒内做出；安全回血窗口判断准确率 > 90%。

**每日时长**：10 分钟。

## 六、每日练习模板

**总时长：35-40 分钟**

```
[5分钟] 热身 — 对空反应训练
  设 Dummy 随机跳，练 623K 对空
  目标：进入"看到跳就 Scratch Wheel"的状态

[5分钟] 确认连招训练
  2MK / 5MK 随机防御确认
  命中→连招，被防→停手
  练到手自动反应

[5分钟] 惩罚连招训练
  防升龙/DI → 最大伤害惩罚
  附加决策：SA1 收尾还是省槽回血？
  练到惩罚+决策同时完成

[10分钟] 起攻 + SA2 回血时机训练
  击倒→判断→压制 or 回血
  不同血量场景下的 SA 决策练习
  安全回血窗口判断

[10分钟] 角色专属技能
  Rhino Horn 距离感
  Spin Scythe 版本区分
  对空→回血→控制 循环练习
  SA 决策场景模拟

[实战后复盘]
  回放中检查：
```
