# Manon 专属训练指南

## 一、角色定位与核心打法

Manon 是 SF6 中的柔道+芭蕾投技角色，拥有独特的 **Medal 等级系统**。投中次数越多，投技伤害越高，且 Medal 等级**跨回合、跨局不重置**。Manon 的核心是"前期积累 Medal，后期用恐怖的投技伤害碾压"。

**核心思路**：
- Medal 系统是 Manon 的灵魂，每次 Manege Dore (360+K) 或 SA1 (236236K) 命中 → Medal +1
- Medal Lv5 时投技伤害翻倍，一次投技几乎打掉对手半血
- 前期（回合 1-2）优先用各种手段升 Medal，即使牺牲一些伤害
- SA1 (236236K) 既是超杀又能升 Medal，是最高效的升级手段
- Hitbox 360 输入方法与 Zangief 相同，但按 K（脚键）

**距离策略**：
- 全屏：没有飞道具，需要前走/DR 接近
- 中距离：2MK 牵制，寻找 DR 确认机会
- 近距离：投/打二择，优先投技升 Medal

**Medal 等级伤害表**：
```
Lv1: 基础伤害
Lv2: 伤害 +~20%
Lv3: 伤害 +~40%
Lv4: 伤害 +~60%
Lv5: 伤害 +~80-100% (恐怖！)

关键: Medal 跨回合/跨局不重置！
= 比赛越打投技越痛
```

## 二、核心连招

### BnB连招（必练）

```
1. 5LP > 5LK > 360+LK (Manege Dore)
   用途：Tick Throw 升 Medal！
   说明：最基础的投技连携，命中 Medal+1
   优先级：前期最重要的连招

2. 2LK > 2LP xx 236236K (SA1 Arabesque)
   用途：轻确认接 SA1 升 Medal！
   说明：消耗 1 槽超杀，但能升 Medal 非常值得

3. 2MP > 5MP xx 236HK (Rond-point)
   用途：中拳确认高伤害路线（不升 Medal）
   说明：不需要升 Medal 时的纯伤害路线
```

### Drive Rush连招

```
1. 2MK xx DR > 5MP > 6MP xx 236236K (SA1)
   用途：DR 确认接 SA1 升 Medal！
   说明：最推荐的 DR 路线，伤害+升 Medal 两不误

2. 2MK xx DR > 5MP > 6MP xx 236HK (Rond-point)
   用途：DR 标准伤害路线（无超杀时）
   说明：没有超杀槽时的 DR 标准路线

3. 2MK xx DR > 5MP > 6MP xx 360+LK (Manege Dore)
   用途：DR 接投技升 Medal
   说明：消耗 Drive 但不消耗超杀槽也能升 Medal
```

### 惩罚连招

```
1. 5HP > 6MP xx 236HK (Rond-point)
   用途：标准惩罚连招
   说明：大破绽时用 5HP 起手追求伤害

2. (PC) 5HK > 5HP > 6MP xx 236236K (SA1)
   用途：PC 最大伤害 + 升 Medal
   说明：升龙被防后的最佳选择

3. (PC) 5HK > 5HP > 6MP xx 236236P (SA3)
   用途：PC 最大绝对伤害
   说明：满 3 槽时的终极惩罚
```

### 跳入连招

```
1. j.HP > 5HP > 6MP xx 236HK (Rond-point)
   用途：跳入标准伤害

2. j.HP > 5HP > 6MP xx DR > 5HP > 6MP xx 236HK
   用途：跳入 DR 延伸高伤害
   说明：消耗 3 格 Drive 换取极高伤害

3. j.HP > 5HP > 6MP xx 236236K (SA1)
   用途：跳入接 SA1 升 Medal
```

### 超杀收尾

```
1. ...xx 236236K (SA1 Arabesque)
   用途：前冲投超杀，命中 Medal+1！
   场景：任何连招收尾都优先考虑 SA1（前期升 Medal 阶段）
   说明：这是 Manon 最重要的超杀用法

2. ...xx 236236P (SA3 Pas de Deux)
   用途：Level 3 最大伤害
   场景：Medal 已满或需要击杀时

3. DI (墙贴) > 5HP > 6MP xx 236236K (SA1)
   用途：DI 墙贴后接 SA1 升 Medal
```

## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心**：投技内战的关键是谁的投/打读心更准，Manon 的优势在于 Medal 系统的成长性。

- **Medal 策略**：
  - 前期即使被打也要积极投，升 Medal 的长期收益远大于短期伤害
  - SA1 确认接投比裸投更安全（SA1 有超杀动画确认）
- **vs Zangief**：
  - 他的 SPD 伤害一开始就很高，不要硬换血
  - 中距离用 2MK 牵制，不给他贴身 SPD 的机会
  - 读到他 SPD → 后跳 j.HP 惩罚（投技打空硬直大）
  - 你的 Medal 升高后单次投技伤害会超过他的 SPD
- **vs 镜像 (Manon)**：谁先升 Medal 谁占优，积极用 SA1
- **vs Lily**：注意她的 Condor Spire 突进，2MK 打点控制距离

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心**：防住速攻压制后反击，利用投/打二择和 Medal 系统的成长性扭转局面。

- **防守策略**：
  - 蹲防为主，被压制时 Drive Reversal 推开
  - En Haut (623P) OD 无敌可以在被压时翻身（但风险高）
  - Perfect Parry 对手的固定套路后接最大惩罚
- **反击后升 Medal**：
  - 防住对手压制后，能投就投（升 Medal）
  - 确认连招尽量收 SA1（升 Medal）
  - 速攻角色血量低，Medal Lv3+ 后一次投就打掉他们大量血量
- **对空**：
  - 623K (Renverse) 对空投，命中还能升 Medal！
  - 速攻角色跳多，Renverse 对空成功 = 升 Medal + 伤害 + 位置
- **心理战**：
  - Medal 高了之后对手会极度怕被投
  - 利用这个心理优势打更多打击确认

### vs 远程角色 (JP/Guile/Dhalsim)

**核心**：最困难的对局。Manon 没有飞道具也没有 armor 突进，需要极度耐心接近。

- **接近方法**：
  - 前走 + Drive Parry 弹反飞道具（恢复 Drive Gauge）
  - Drive Rush 穿过飞道具间隔
  - DI 吸收飞道具接近（中距离有效）
  - 214K (Demi-Grand-Rond) 后旋踢有回避效果，可以穿过部分飞道具
- **具体对策**：
  - vs JP：注意地面陷阱，从上方跳入时小心对空
  - vs Guile：Sonic Boom 间隔 Parry + DR 接近；不要中距离跳
  - vs Dhalsim：耐心前走，贴身后猛攻
- **贴身后策略**：
  - 一旦接近绝不松手，投/打二择猛攻
  - 优先用投技/SA1 升 Medal
  - 接近一次可能只有一次投技机会，一定要抓住
- **Medal 策略调整**：
  - 对远程角色接近困难，每次接近的投技机会更珍贵
  - 即使用 SA1 消耗超杀槽也值得升 Medal

### vs 重型角色 (Marisa/Honda)

**核心**：利用 Manon 的速度优势和 Medal 系统的成长性。

- **vs Marisa**：
  - Marisa 的 armor 可以被投技破解！读到她蓄力 Gladius → 走上去投
  - 中距离不要和她比打击，她的单次伤害更高
  - 利用 Manon 的速度优势快进快出
  - Medal 升高后投技伤害不输 Marisa 的打击伤害
- **vs Honda**：
  - LP Headbutt (-2) 基本安全，准备好下次进攻
  - HP Headbutt 被防后惩罚：5HP > 6MP xx 236HK
  - 百裂张手压制时 Drive Reversal 推开
  - Oicho 指令投是蓄力技，对方蓄力时间就是你接近的机会
  - 角落 Honda 特别危险，不要被逼到角落

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心**：经典投技 vs 波动拳对决，耐心接近后用投/打择和 Medal 系统取胜。

- **反飞道具**：
  - Drive Parry 弹反波动拳 + DR 接近
  - 全屏安全跳跃越过飞道具
  - DI 吸收飞道具接近
- **接近后策略**：
  - 贴身后投/打二择，优先投技升 Medal
  - 读到升龙 → 蹲防 → 最大惩罚 + SA1 收尾升 Medal
  - 对手不敢出升龙时 → 投技更容易命中
- **具体注意**：
  - vs Ryu：最传统对局，耐心接近
  - vs Ken：注意 DR 接近速度，准备好 623P OD 反击
  - vs Luke：他的 5MP 范围远，2MK 打点后 DR 接近
  - vs Akuma：血量最低，Medal Lv3+ 后一次投技伤害极高
- **Medal 心理优势**：
  - Medal Lv4-5 时对手极度紧张，"被投一次就半血"的压力
  - 利用这个心理优势打更多打击确认

## 四、训练模式练习法

### 练习1：对空反应训练

**设置**：Dummy 设为随机跳入 (Record: 前跳 j.HP / 前跳空跳 / 不跳)

**练习内容**：
1. 中距离站立，Dummy 随机跳
2. 看到跳入 → 623K (Renverse) 对空投（命中 Medal+1！）
3. 或 → OD 623P (En Haut) 无敌对空
4. 没跳 → 2MK 牵制 / 前走接近

**目标**：连续 10 次对空成功率 70% 以上

**要点**：
- Renverse 对空投命中升 Medal，是 Manon 独特的对空收益
- 对空距离和时机很重要，太远或太晚会打空
- OD En Haut 有无敌但消耗 2 格 Drive

### 练习2：确认连招训练

**设置**：Dummy 设为随机防御 (Block Random)

**练习内容**：
1. 2LK > 2LP > 确认命中 → xx 236236K (SA1) 升 Medal
2. 2LK > 2LP > 确认防御 → 停手或走上去投
3. 2MK > 确认命中 → xx DR > 5MP > 6MP xx SA1 升 Medal
4. 2MK > 确认防御 → 停手

**目标**：命中确认准确率 70% 以上

**要点**：
- Manon 的确认连招要优先接 SA1 升 Medal
- 轻确认窗口约 15-16 帧，先练慢后练快
- 没有超杀槽时改接 236HK (Rond-point) 追求伤害

### 练习3：惩罚训练

**设置**：录制 Dummy 出各种可惩罚招式

**练习内容**：
1. Dummy 升龙被防 → (PC) 5HK > 5HP > 6MP xx SA1 (升 Medal)
2. Dummy DI 被防 → 5HP > 6MP xx SA1
3. Dummy 不安全招式 → 5HP > 6MP xx 236HK
4. 小破绽 (-4~-5) → 5LP > 5LK > 360+LK (投技升 Medal)

**目标**：大破绽 100% 出最大惩罚 + 升 Medal

**要点**：
- 惩罚连招收尾优先 SA1 升 Medal（前期）
- Medal Lv3+ 后可以选择纯伤害路线 (SA3)
- 小破绽也要抓住机会投技升 Medal

### 练习4：起攻/压制训练

**设置**：Dummy 设为起身后随机行动

**练习内容**：
1. 投技命中后 → 前走 meaty 2LK → 对手防 → tick throw (360+LK) 升 Medal
2. 投技命中后 → 前走 meaty 2LK → 对手跳 → 623K (Renverse) 对空 升 Medal
3. Rond-point 命中后 → 前走 meaty 5MP → 帧陷阱
4. 角落击倒后 → DR meaty → 投/打二择

**目标**：每种起身选项都有对应方案，且尽量升 Medal

**要点**：
- Manon 的起攻核心是"升 Medal"
- 对手防守 → 投（升 Medal）
- 对手跳 → Renverse 对空投（升 Medal）
- 对手按键 → Counter Hit 确认 → SA1（升 Medal）
- 无论对手怎么选，都有升 Medal 的路线

### 练习5：角色专属技能训练 — 360 输入与 Medal 管理

**Hitbox 360 输入训练（按 K）**：

```
步骤1：原地 Manege Dore
站立状态 → 6 > 2 > 4 > 8+K
练到 10/10 成功，不跳起

步骤2：蹲防起始 360+K（推荐方法）
蹲防(↓+←) → 松开↓ → 按→ → 按↑+K
即: 1(下后) > 4(后) > 6(前) > 8+K(上+脚)
练到 10/10 成功

步骤3：Tick Throw 360+K
2LK(蹲轻脚，按着↓) → 恢复帧中: 6 > 8 > 4 + K
练到连续 5 次不失误

步骤4：DR 360+K
2MK xx DR(66) → 冲刺中: ↓ > ← > ↑ + K
练到连续 5 次不失误

步骤5：SA1 (236236K) 替代练习
SA1 比 360+K 更容易输入（双向前+K）
在不确定时用 SA1 替代指令投
SA1 同样升 Medal！
```

**Medal 升级路线模拟**：
```
模拟一局游戏的 Medal 升级：
回合1: 2LK > 2LP xx SA1 (Lv1→Lv2)
回合2: tick throw 360+K (Lv2→Lv3) + SA1 (Lv3→Lv4)
回合3: 投技已经非常痛，对手心理压力巨大
目标: 练习一整局里的 Medal 升级节奏
```

- 5HP > 6MP xx 236HK x10
- 2MK xx DR > 5MP > 6MP xx SA1 x10

- Dummy Random Block

- Dummy 随机跳入
- 623K (Renverse) 对空投升 Medal
- OD 623P (En Haut) 对空
