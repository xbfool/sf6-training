# Blanka 专属训练指南

## 一、角色定位与核心打法

Blanka 是 SF6 中的野性角色，以不可预测的移动 + 滚球攻击 + Blanka-chan Bomb（玩偶炸弹）设置为核心。MenaRD 使用 Blanka 夺得 EVO 2025 冠军，证明了该角色在最高水平赛场的实力。

**核心思路**: 利用滚球突进制造压力 + 电击近身防御 + 玩偶炸弹设置起攻/控制空间。Blanka 的打法核心是"不可预测"——通过多变的移动方式和时机让对手无法应对。

**最佳距离**: 中近距离（滚球突进范围 + 电击防御范围）。远距离放玩偶控制空间。

**关键招式总览**:

| 招式 | 指令 | 用途 |
|------|------|------|
| Rolling Attack | [4]6P | 水平滚球突进 |
| Vertical Rolling | [2]8K | 垂直滚球对空 |
| Backstep Rolling | [4]6K | 后退再前冲 |
| Electric Thunder | 236P | 电击，近身防御/压制 |
| Blanka-chan Bomb | 214P | 放置玩偶炸弹 |
| Wild Assault | 214K | 前冲攻击 |
| SA1 Shout of Earth | 236236P | 电击超杀 |
| SA3 | 236236K | Level 3 超杀 |

**分距离战术**:
- **远距离**: 214P（Blanka-chan Bomb）放置玩偶控制空间，214HP 远距离版
- **中距离**: [4]6P（Rolling Attack）突进，[4]6K（Backstep Rolling）出其不意
- **近距离**: 236P（Electric Thunder）电击近身防御，轻攻击压制接投/打二择
- **对空**: [2]8K（Vertical Rolling）垂直滚球

## 二、核心连招

### BnB连招（必练）

```
2LP > 2LP > 5LP xx 236LP              轻确认接电击，最基础
2MK xx [4]6LP                          蹲中脚接轻滚球，安全打点
5MP > 5MK xx 236MP                     中拳确认接中电击
2MK xx 236LP                           蹲中脚接轻电击（不需要蓄力版本）
```

### Drive Rush连招

```
2MK xx DR > 5MP > 5HP xx [4]6HP        标准DR接重滚球，高伤害
2MK xx DR > 5MP > 5HP xx 236HP         DR接重电击
2MK xx DR > 5MP > 2MP xx [4]6MP        DR中拳确认接滚球
```

### 惩罚连招

```
对手DP被防:
5HP xx OD236P(OD Electric) > [4]6HP     OD电击接重滚球最大伤害
5HP xx DR > 5MP > 5HP xx [4]6HP         DR延伸惩罚

DI被防/墙贴:
DI(hit) > 5HP xx OD236P > [4]6HP        DI高伤害
5HP xx DR > 5MP > 5HP xx [4]6HP > SA3   超杀收尾

对手投技打空:
5HP xx [4]6HP                            重拳接重滚球
```

### 跳入连招

```
j.HP > 5HP xx [4]6HP                    跳入重拳接重滚球
j.HP > 5MP > 5HP xx 236MP               跳入确认接电击
j.HP > 2MK xx DR > 5MP > 5HP xx [4]6HP  跳入DR最大伤害
交叉 j.MK > 2LP > 2LP > 5LP xx 236LP   交叉跳确认
```

### 超杀收尾

```
2MK xx DR > 5HP xx OD236P > [4]6HP > SA3   DR最大伤害超杀收尾
... xx [4]6HP > SA1(Shout of Earth)         滚球接SA1
DI(墙贴) > 5HP xx OD236P > [4]6HP > SA3    DI墙贴最大伤害
确认命中 > 236236P(SA1)                     电击超杀收尾
```

## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心**: 利用滚球机动性保持距离，玩偶炸弹控制空间。

- **中远距离**: 214HP（远距离玩偶）放置在对手前进路线上，限制接近
- **对手接近**: [4]6LP/MP（Rolling Attack）滚球拉开距离，或 [4]6K（Backstep Rolling）后退再冲
- **近距离**: 236P（Electric Thunder）电击推开对手，不要和投技角色在近距离对耗
- **对手跳入**: [2]8K（Vertical Rolling）垂直滚球对空
- **被贴身**: Drive Reversal 推开，后跳逃脱投技范围
- **注意**: Zangief 的 DI 有 armor，滚球可能被吸收反打；用多段电击（236P）消耗 armor

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心**: 不可预测性对抗速攻角色的节奏。利用滚球和玩偶打乱对手节奏。

- **中距离**: [4]6LP/MP（Rolling Attack）突进打断对手节奏
- **对空**: [2]8K（Vertical Rolling）对空，速攻角色跳跃频繁
- **被压制**: 236P（Electric Thunder）近身电击打断压制（注意帧数，不是所有情况都能打断）
- **玩偶运用**: 214P 放玩偶在自己身前，对手冲进来被炸
- **Drive Reversal**: 被连续压制时花 2 格推开
- **注意**: 滚球被防是负帧可以被惩罚，不要无脑滚；选择时机很重要

### vs 远程角色 (JP/Guile/Dhalsim)

**核心**: 利用滚球快速接近，玩偶限制对手远程输出。

- **对付飞道具**: [4]6P（Rolling Attack）可以穿过飞道具间隔突进（注意时机）
- **214K（Wild Assault）**: 前冲攻击有 armor，可以吸收一发飞道具接近
- **玩偶运用**: 214P 放在中距离，限制对手放弹的安全空间
- **接近后**: 一旦贴身猛攻不给对手重新拉开的机会，236P 电击保持压力
- **vs Guile 特殊**: 滚球可以从 Sonic Boom 下方穿过（LP版低姿态），但被读到会被 Flash Kick
- **vs JP 特殊**: 注意 JP 的传送和陷阱，不要冲进 Amnesia 陷阱范围
- **vs Dhalsim 特殊**: 滚球可以打到 Dhalsim 伸长的肢体

### vs 重型角色 (Marisa/Honda)

**核心**: 利用机动性绕开正面硬碰硬。

- **滚球突进**: [4]6LP 轻滚球快进快出，不要用重滚球（容易被 armor 反打）
- **电击多段**: 236P 多段电击可以消耗 Marisa 的 armor
- **对 Honda**: Honda 头槌和滚球可能交叉，注意时机；用 [2]8K 对空 Honda 的跳入
- **玩偶**: 214P 放在中距离限制重型角色前进
- **注意**: 不要和重型角色比伤害，利用速度优势快打快走

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心**: 滚球穿弹接近 + 玩偶压制对手放弹空间。

- **穿弹突进**: [4]6LP（Rolling Attack）在对手放波动拳的恢复帧穿过去命中
- **214K（Wild Assault）**: armor 吸收飞道具后接近
- **中距离**: 2MK 牵制，不要在中距离和这些角色比 footsies（他们的 2MK 更稳定）
- **玩偶运用**: 214P 放在中距离，限制对手放弹频率
- **对空**: [2]8K（Vertical Rolling）对空，这些角色偶尔会跳
- **注意**: Ken/Akuma 有无敌升龙，被压制时他们可能 DP 翻身，准备好防御后惩罚

## 四、训练模式练习法

### 练习1：对空反应训练

**目标**: 稳定用 [2]8K（Vertical Rolling）对空

**设置方法**:
1. 训练模式 > Dummy 设置为"随机跳入"
2. Blanka 保持蹲蓄力（↓），随时准备 Vertical Rolling
3. Dummy 跳入时反应出 [2]8MK/HK

**练习要点**:
- 保持 ↓ 蓄力，看到跳跃 → [2]8K
- 注意不同距离：近距离用 LK 版，远距离用 HK 版
- OD 版（[2]8KK）有无敌，紧急时使用

**通过标准**: 连续10次随机跳入中对空成功 8 次以上

### 练习2：确认连招训练

**目标**: 2MK 命中确认接滚球/电击

**设置方法**:
1. Dummy 设置为"随机防御"
2. 出 2MK，命中时取消接 [4]6LP 或 236LP
3. 被防时不取消

**练习要点**:
- 2MK 命中 → [4]6LP（Rolling Attack）或 236LP（Electric Thunder）
- 2MK 被防 → 不取消，保持安全
- 进阶：2MK 命中 → DR > 5MP > 5HP xx [4]6HP

**通过标准**: 30次中正确确认 25 次以上

### 练习3：惩罚训练

**目标**: 对手 DP/DI 被防后最大伤害

**设置方法**:
1. Dummy 录制：起身后出升龙拳
2. 你防住后 5HP xx OD236P > [4]6HP

**练习要点**:
- 标准惩罚：5HP xx OD236P > [4]6HP
- DR 惩罚：5HP xx DR > 5MP > 5HP xx [4]6HP
- 超杀惩罚：5HP xx OD236P > [4]6HP > SA3

**通过标准**: 连续10次 DP 被防后全部成功惩罚

### 练习4：起攻/压制训练

**目标**: 击倒后 Blanka-chan Bomb 设置起攻

**设置方法**:
1. 击倒对手后立刻 214P（放玩偶在对手起身位置）
2. 玩偶掩护下进行投/打二择

**练习要点**:
- **玩偶起攻**: 击倒 > 214LP（近距离放玩偶）> 前走 > 对手起身时玩偶爆炸 > 投/打二择
- **远距离玩偶**: 214HP > 玩偶在远距离控制空间
- **交叉跳**: 击倒后跳入交叉（j.MK 打到对手身后）> 玩偶爆炸 > 连招
- **帧陷阱**: 5LP > 5LP > 微延迟 > 5MP（对手按键被 Counter）

**通过标准**: 连续10次击倒后玩偶设置 + 起攻成功

### 练习5：角色专属技能训练 — 滚球运用与不可预测性

**目标**: 掌握不同版本滚球的运用时机

**训练A — Rolling Attack 距离感**:
1. [4]6LP/MP/HP 三个版本分别练习，感受距离和速度差异
2. LP 版近距离安全、HP 版远距离高伤害但恢复慢
3. 练习在对手放招的恢复帧使用滚球惩罚

**训练B — Backstep Rolling 出其不意**:
1. [4]6K（Backstep Rolling）先后退再前冲
2. 练习在对手预判你后退时使用，打出意外命中
3. 配合前滚球交替使用，制造不可预测性

**训练C — Electric Thunder 近身运用**:
1. 236LP/MP/HP 三个版本的帧数和范围差异
2. 近距离压制串中插入电击
3. OD 电击（236PP）的追加连招

**训练D — Blanka-chan Bomb 位置控制**:
1. 214LP（近距离放）/ 214MP（中距离放）/ 214HP（远距离放）
2. 不同击倒技后选择合适版本的玩偶
3. 玩偶位置覆盖对手起身范围的精确度

**通过标准**: 滚球三个版本各 10 次无失误，玩偶设置位置准确 8/10

## 六、每日练习模板

```
总时间: 30-40 分钟

【热身】5 分钟
- [4]6LP/MP/HP x 5 each（三个版本滚球手感）
- [2]8MK x 10（垂直滚球对空）
- 236LP x 10（电击手感）
- 214LP/MP/HP x 3 each（三个版本玩偶位置）

【连招练习】10 分钟
- 2LP > 2LP > 5LP xx 236LP x 10（轻确认基础）
- 2MK xx [4]6LP x 10（蹲中脚接滚球）
- 2MK xx DR > 5MP > 5HP xx [4]6HP x 10（DR 连招）
- 5HP xx OD236P > [4]6HP x 5（惩罚连招）
- 超杀收尾 x 5

【确认训练】5 分钟
- Dummy 随机防御，2MK 确认 x 20
  - 命中 → [4]6LP 或 DR 连招
  - 被防 → 停手

【对空训练】5 分钟
- Dummy 随机跳入，[2]8K 对空 x 20
- 目标: 成功率 80%+

【玩偶起攻训练】10 分钟
- 击倒后 214LP 玩偶设置 x 10
- 玩偶掩护下投/打二择 x 10
- 交叉跳 + 玩偶组合 x 5
- 远距离玩偶空间控制 x 5

【实战应用】5-10 分钟
- Rank 或 Casual 实战 2-3 局
- 专注: 玩偶设置频率、对空成功率、滚球命中率
- 记录: 玩偶设置次数、对空次数、滚球被惩罚次数
```
