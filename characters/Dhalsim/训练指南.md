# Dhalsim 专属训练指南

## 一、角色定位与核心打法

Dhalsim 是 SF6 中操作难度最高的角色，拥有全游戏最远的普通技范围（伸长四肢）。超远距离控制 + Yoga Fire 飞道具 + Yoga Teleport 传送 + Yoga Float 悬停下压，构成独特的"远距离打点"体系。

**核心思路**: 在超远距离用伸长的四肢（5HP/5HK/2MK）不断打点消耗对手 → Yoga Fire 控制空间 → 对手接近到中距离时必须立刻拉远或传送逃脱 → 击倒后用慢火球掩护起攻。

**最佳距离**: 超远距离（全屏或接近全屏）。中距离是 Dhalsim 的死亡区间，必须避免停留。近身更是极度危险。

**关键招式总览**:

| 招式 | 指令 | 用途 |
|------|------|------|
| Yoga Fire | 236P | 飞道具 |
| Yoga Arch | 236K | 向上抛射火焰 |
| Yoga Flame | 214P | 近距离喷火 |
| Yoga Float | 空中214K | 空中悬停 |
| Yoga Teleport | 623PPP/KKK | 前方/后方传送 |
| Drill Kick | j.2MK/HK | 空中下压踢 |
| SA2 Yoga Sunburst | 214214P | 设置型火球 |
| SA3 | 236236K | Level 3 超杀 |

**分距离战术**:
- **超远距离（舒适区）**: 5HP/5HK/2MK 伸长打点 + 236P（Yoga Fire）控制空间
- **中距离（危险区）**: 不要在这里停留！立刻用前走拉远 或 623PPP/KKK 传送逃脱
- **近距离（极危险）**: 214P（Yoga Flame）近身喷火推开、623PPP 传送逃脱、Drive Reversal 脱压
- **空中**: Yoga Float（空中214K）悬停 → Drill Kick（j.2MK/HK）下压 或 空中火球

## 二、核心连招

### BnB连招（必练）

```
2LP > 2LP xx 236LP                   轻确认接 Yoga Fire，最安全的确认
5MP(近) > 5MK xx 236MP               近距离中拳确认接中火球
2MK(远距离) 单发打点                   远距离伸长蹲中脚，不需要取消
5HP(远距离) 单发打点                   远距离伸长站重拳牵制
```

### Drive Rush连招

```
2MP > 5MP xx DR > 5MP > 5HP xx 214LP    DR延伸接近距离火焰
2MK xx DR > 2MP > 5MP xx 236MP          DR从远距离延伸
2LP > 2LP xx DR > 5MP > 5MK xx 236MP    轻确认DR延伸
```

### 惩罚连招

```
对手DP被防:
5HP > 5MP > 5MK xx 236HP                重拳起手最大伤害
5HP xx DR > 5MP > 5HP xx 214MP          DR延伸惩罚

DI被防:
5HP > 5MP xx 236HP > SA1                超杀收尾

近距离惩罚:
5MP > 5MK xx 214MP                      中拳接火焰
```

### 跳入连招

```
j.2MK(Drill) > 2LP > 5MP xx 236MP      Drill Kick 起手确认
j.2HK(Drill) > 5MP > 5MK xx 214MP      重 Drill 起手高伤害
j.HP > 5MP > 5MK xx 236MP              空中重拳跳入
j.2MK(Drill) > 2LP > 5MP xx DR > 5HP xx 214MP  Drill 接 DR 延伸
```

### 超杀收尾

```
j.2MK(Drill) > 2LP > 5MP xx 236236P(SA1)    Drill 接 SA1

击倒后 214214P(SA2 Yoga Sunburst) 设置       设置型火球控场
```

## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心**: Dhalsim 对投技角色有极大优势，因为伸长肢体在全屏就能打到对手。

- **超远距离**: 5HP/5HK/2MK 不断打点，Zangief/Manon 无法接近
- **236P（Yoga Fire）**: 持续放弹增加接近难度
- **对手跳弹**: Yoga Arch（236K）抛射覆盖空中，或反应 Drill Kick 下压交叉
- **对手DR接近**: 后跳 Drill Kick 或 623KKK（后方传送）逃脱
- **被贴身**: Drive Reversal 推开，然后 623PPP 传送拉远重建距离
- **关键**: 永远不要让 Zangief 贴身！一旦被 SPD 投到就是巨大伤害
- **Lily 特殊**: Lily 的 Condor Spire 突进速度快，注意在中远距离预判打 5HP 惩罚

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心**: 保持最远距离，速攻角色接近速度快，Dhalsim 必须提前做好准备。

- **超远距离**: 5HP/2MK 打点保持距离，236P 控制前进路线
- **对手跳入**: Dhalsim 缺乏传统无敌对空！236K（Yoga Arch）覆盖空中，或空中 j.MP/j.MK 空对空
- **对手冲进中距离**: 立刻 623KKK（后方传送）逃脱！中距离是死亡区间
- **被压制**: Drive Reversal 脱压 → 传送拉远
- **角落**: 极其危险！用 623PPP（前方传送穿过对手身后）逃脱角落
- **注意**: Cammy 的速度极快，不要在中距离出慢招；Kimberly 传送后要准备应对

### vs 远程角色 (JP/Guile/Dhalsim)

**核心**: 远程对轰，Dhalsim 的伸长肢体比飞道具更有范围优势。

- **vs JP**: 5HP 可以在 JP 放弹时打到他；注意 JP 的 Torbalan（214P）地面灵力可以打到 Dhalsim 伸长的脚。传送需要小心 JP 的 Embrace 对空
- **vs Guile**: Sonic Boom 频率高，Dhalsim 需要用 236P 对弹+5HP/5HK 远距离打点穿过弹。不要跳——Guile 的 Flash Kick 对空极强
- **vs 镜像 Dhalsim**: 谁先建立空中控制（Yoga Float + Drill Kick）谁占优势
- **通用**: 伸长肢体有被对手打到的风险（判定框在伸长部分），注意不要在对手有准备时出慢招

### vs 重型角色 (Marisa/Honda)

**核心**: 全屏打点消耗，重型角色接近困难。

- **超远距离**: 5HP/5HK 不断打点，Marisa/Honda 接近速度慢
- **236P（Yoga Fire）**: 持续放弹控制
- **Honda 头槌**: 预判用 5HK 远距离打断（伸长脚可以打到头槌中的 Honda）
- **Marisa armor**: 注意不要在中距离出慢的伸长招式，Marisa 的 armor 可以吸收后惩罚
- **对手DI**: 后跳或传送逃脱，不要硬接
- **关键**: 保持最远距离，利用 Dhalsim 独有的超远范围优势

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心**: 伸长肢体可以在飞道具间隔打到对手。但要注意升龙对空。

- **超远距离**: 5HP 在对手放波动拳时打到他（伸长手臂穿过飞道具前方打到本体）
- **236P 对弹**: Yoga Fire 与对方飞道具对消
- **Yoga Float + Drill Kick**: 空中下压绕过飞道具，但注意对手的升龙对空
- **中距离**: 不要停留！这些角色在中距离很强（2MK/5MK/DR），Dhalsim 必须保持远距离
- **注意**: Ken/Akuma 的 DR 速度快，一旦被冲进来就很危险；及时传送逃脱

## 四、训练模式练习法

### 练习1：对空反应训练

**目标**: 用多种方式应对跳入（Dhalsim 没有标准无敌对空）

**设置方法**:
1. Dummy 设置为"随机跳入"
2. Dhalsim 站在远距离
3. 根据距离选择不同对空方式

**对空选择**:
- 远距离跳入 → 5HK（伸长脚）提前打
- 中距离跳入 → 236K（Yoga Arch）抛射覆盖
- 近距离跳入 → 空中 j.MP 空对空 或 后跳 j.MK
- 所有距离 → 623KKK（后方传送）逃脱

**通过标准**: 连续10次随机跳入中成功应对 7 次以上

### 练习2：确认连招训练

**目标**: 区分远距离打点和近距离确认

**设置方法**:
1. Dummy 设置为"随机防御"
2. 远距离：5HP/2MK 单发打点（不需要取消）
3. 近距离：2LP > 2LP 确认接 236LP

**练习要点**:
- 远距离 5HP 打中 → 不取消，收招安全（远距离伸长招式无法取消）
- 近距离 2LP > 2LP 命中 → 取消 236LP
- 进阶：近距离 5MP > 5MK 命中 → 236MP 或 DR 延伸

**通过标准**: 30次中正确判断距离并选择合适应对 25 次以上

### 练习3：惩罚训练

**目标**: 对手可惩罚招式后的最大伤害

**设置方法**:
1. Dummy 录制：出升龙拳/DI
2. 你防住后最大伤害惩罚

**练习要点**:
- 5HP > 5MP > 5MK xx 236HP — 标准惩罚
- 5HP xx DR > 5MP > 5HP xx 214MP — DR 延伸
- DI 被防 → 5HP > 5MP xx 236HP > SA1 — 超杀收尾

**通过标准**: 连续10次被防后全部成功惩罚

### 练习4：起攻/压制训练

**目标**: 击倒后慢火球掩护 + 传送左右择

**设置方法**:
1. 击倒对手后立刻 236LP（慢 Yoga Fire）
2. 火球和 Dhalsim 同时压向对手起身位置
3. 或用 SA2（Yoga Sunburst）设置火球

**练习要点**:
- **慢火球起攻**: 击倒 > 236LP > 前走跟随火球 > 火球掩护下投/打二择
- **传送起攻**: 击倒 > 623PPP（传送到对手身后）> 左右二择
- **SA2 设置**: 击倒 > 214214P（Yoga Sunburst 设置）> 火球在场时对手被迫应对
- **Drill Kick 压制**: Yoga Float > Drill Kick 下压 > 落地后续攻

**通过标准**: 连续10次击倒后成功建立起攻

### 练习5：角色专属技能训练 — 距离管理与空中机动

**目标**: 掌握 Dhalsim 的超远距离打点 + 空中悬停下压

**训练A — 远距离打点**:
1. Dummy 设置为前走
2. 用 5HP/5HK/2MK 在最远距离不断打点
3. 目标：让 Dummy 永远无法接近到中距离
4. 一旦 Dummy 接近中距离 → 623KKK 传送逃脱

**训练B — Yoga Float + Drill Kick**:
1. 跳跃 > 空中 214K（Yoga Float 悬停）> j.2MK/j.2HK（Drill Kick 下压）
2. 练习悬停高度和 Drill Kick 时机
3. Drill Kick 命中后接地面连招（2LP > 5MP xx 236MP）
4. 练习不同角度的 Drill Kick

**训练C — 传送逃脱**:
1. Dummy 设置为持续攻击
2. 练习 623PPP（前方传送）和 623KKK（后方传送）的选择
3. 传送后立刻建立远距离重新打点
4. 被角落逼住时 → 623PPP 穿过对手逃脱

**通过标准**: 保持远距离控制 30 秒不被接近，Drill Kick 命中后接连招 8/10 成功

```

- 2MK x 10（远距离蹲中脚）

- Dummy 随机跳入，选择对空方式 x 15

【实战应用】5-10 分钟
- Rank 或 Casual 实战 2-3 局
- 专注: 距离管理（是否停留在中距离？）、传送逃脱时机
- 记录: 被接近次数、传送逃脱成功次数、Drill Kick 命中次数
```
