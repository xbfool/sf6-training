# Marisa 专属训练指南

## 一、角色定位与核心打法

Marisa 是 SF6 中的重型打击手，拥有全游戏最高的单次打击伤害。多招式自带 **Armor（霸体）**，蓄力招式可以强行突破对手防线。打法风格是"用伤害差碾压对手"。

**核心思路**：
- 单次伤害全游戏最高，一个 combo 可以打掉对手 40-50% 血量
- Gladius (236P) 蓄力版有 armor，可以霸体突进吸收对手攻击后反击
- Phalanx (214P) 有 armor，是优秀的反击/对空工具
- Scutum (214K) 盾牌架势可以吸收攻击，派生 P 反击或 K 前冲
- Tonitrus (623P) OD 版有无敌，是唯一的无敌反击手段
- 蓄力 Gladius meaty 起攻是角落最强压制手段之一

**距离策略**：
- 全屏：没有飞道具，需要前走/DR 接近
- 中距离：Marisa 的最佳距离！5HP/2MK 牵制，Gladius 突进
- 近距离：5MP/2MP 压制，投/打二择，蓄力 Gladius 给对手巨大压力

## 二、核心连招

### BnB连招（必练）

```
1. 2LK > 2LP xx 236LP (Gladius)
   用途：轻确认，最基础的连招
   说明：轻攻击确认命中后接 Gladius 收尾

2. 2MP > 5MP xx 236LP (Gladius)
   用途：中拳确认，中距离主力
   说明：2MP 起手距离更远，确认后接 Gladius

3. 5HP xx 236MK (Dimachaerus) > 236LP (Gladius)
   用途：重拳连招，高伤害
   说明：5HP 起手命中确认后追加两个必杀技
```

### Drive Rush连招

```
1. 2MK xx DR > 5HP xx 236MK (Dimachaerus) > 236HP (Gladius)
   伤害：~3200
   用途：DR 高伤害标准路线

2. 5HP xx 214MP (Phalanx) > 236MK (Dimachaerus)
   用途：Phalanx 命中后追击
   说明：Phalanx 的 armor 吸收对手攻击后反击成功时使用

3. 2MK xx DR > 5MP > 5HP xx 236MK > 236HP
   用途：DR 延伸版高伤害
```

### 惩罚连招

```
1. (PC) 5HK > 5HP xx 236MK > 236HP
   用途：Punish Counter 大伤害
   说明：对手升龙被防等大破绽时，PC 浮空后追击

2. 5HP xx 236MK > 236HP
   用途：标准惩罚连招
   说明：-7 以上的破绽都可以用 5HP 起手

3. 5HP xx 236MK > 5MP xx 236MK
   用途：中距离惩罚延伸
```

### 跳入连招

```
1. j.HP > 5HP xx 236MK > 236HP
   用途：跳入标准高伤害

2. j.HP > 5HP xx 236MK > 5MP xx 236MK > SA1
   用途：跳入接超杀最大伤害
   说明：资源充足时的最大化伤害路线
```

### 超杀收尾

```
1. ...xx 214214P (SA2 Megalomania)
   用途：有 armor 的超杀，可以在对方出招时强行打出
   说明：SA2 自带 armor，在读心对手进攻时也可以裸出

2. ...xx 236236K (SA3 Goddess of Hunt)
   用途：Level 3 超杀收尾，最大伤害
   场景：确认连招的终极收尾

3. DI (墙贴) > 5HP xx 214MP > 236MK > SA3
   用途：DI 墙贴后的最大伤害路线
```

## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心**：利用 Marisa 的高伤害打击和 armor 在中距离建立优势。

- **距离控制**：中距离是 Marisa 的主场，用 5HP/2MK 牵制
- **Armor 优势**：
  - 投技角色接近时蓄力 Gladius 236[HP]，对手打键被 armor 吸收
  - 但注意：**投技无视 armor**！不要在对方投技范围内用 armor
- **Scutum 反击**：对手跳入时 214K 吸收后 P 反击
- **vs Zangief**：保持中距离，不要被贴身；他近身后不要用 armor（会被 SPD）
- **vs Manon**：不让她升 Medal，积极用打击压制；读到投时后跳 j.HP 惩罚
- **vs Lily**：注意 Condor Spire 突进，蹲防后用 5MP 反击

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心**：用 armor 和高伤害换血，速攻角色血量低，换血 Marisa 永远赚。

- **Phalanx 反击 (214P)**：
  - 有 armor，对手连续压制时可以读心 214P 反击
  - 命中后接 236MK 追击
  - 但被读到会被投或多段攻击破解
- **Tonitrus (623P) OD 无敌**：
  - OD 版有无敌，是被压制时的翻身手段
  - 消耗 2 格 Drive，不要滥用
- **蓄力 Gladius 压制**：
  - 对手习惯按键压制时，蓄力 Gladius 吸收后反击
  - 速攻角色攻速快但伤害低，Marisa 一次 armor 反击 = 对方好几次攻击的伤害
- **Drive Reversal**：被持续压制时花 2 格推开，比乱用 armor 安全
- **注意**：不要追着速攻角色跑，控制中距离等对手来

### vs 远程角色 (JP/Guile/Dhalsim)

**核心**：利用 Armor 突进穿过飞道具，一旦接近用高伤害速战速决。

- **接近方法**：
  - 蓄力 Gladius 236[HP]：armor 可以吸收一次飞道具后继续前进
  - Drive Parry 弹反飞道具 + DR 前冲
  - DI 吸收飞道具接近（中距离有效）
  - 前走 + 蹲防稳步推进
- **具体对策**：
  - vs JP：蓄力 Gladius 穿飞道具接近，注意避开地面陷阱 (214P)
  - vs Guile：Sonic Boom 间隔用 Gladius 突进，不要中距离跳（Flash Kick）
  - vs Dhalsim：利用 Phalanx armor 穿过远距离攻击，贴身后猛攻不放手
- **贴身后**：Marisa 的高伤害近身战对远程角色极其有利，一次 combo 可以打掉对手大量血量

### vs 重型角色 (Marisa/Honda)

**核心**：重型内战拼伤害效率和读心。

- **vs 镜像 (Marisa)**：
  - 谁先读对谁赢，双方伤害都极高
  - 注意对方的 Scutum 派生，不要被吸收后反击
  - 蓄力 Gladius 对轰时比的是时机
- **vs Honda**：
  - LP Headbutt (-2) 几乎安全，但可以抢帧 5LP
  - HP Headbutt 被防后大惩罚：5HP xx 236MK > 236HP
  - 百裂张手压制时用 Phalanx (214P) armor 反击
  - Scutum (214K) 可以吸收头槌后反击（需要提前读心）
  - 不要在近距离和 Honda 打消耗，利用中距离 Gladius/Dimachaerus 控制节奏

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心**：利用 armor 突破飞道具封锁，近身后用伤害差碾压。

- **反飞道具**：
  - 蓄力 Gladius 236[HP] 穿过飞道具（armor 吸收后继续前进命中）
  - 不要滥用：对方可以读心跳过或投技破解
  - Drive Parry 弹反 + DR 接近更安全
- **对付升龙**：
  - 读到升龙 → 蹲防 → PC 5HK > 5HP xx 236MK > 236HP（巨额伤害）
  - Marisa 的惩罚伤害极高，对方不敢轻易出升龙
- **SA2 Megalomania 反制**：
  - SA2 自带 armor，在读心对方出招时可以裸出
  - 波动拳系在中距离出飞道具时，SA2 可以穿过去命中
- **具体注意**：
  - vs Ryu：中距离 Gladius 突进可以穿过波动拳
  - vs Ken：注意他的 DR 接近速度，准备好 Phalanx 反击
  - vs Luke：他的 5MP 范围远，不要在他的最佳距离停留
  - vs Akuma：血量最低，Marisa 一套可以打掉 60%+，积极进攻

## 四、训练模式练习法

### 练习1：对空反应训练

**设置**：Dummy 设为随机跳入 (Record: 前跳 j.HP / 前跳空跳 / 不跳)

**练习内容**：
1. 中距离站立，Dummy 随机跳
2. 看到跳入 → 214P (Phalanx) 对空（armor 吸收跳入后反击）
3. 或 → OD 623P (Tonitrus) 对空（无敌对空）
4. 没跳 → 5HP/2MK 牵制

**目标**：连续 10 次对空成功率 70% 以上

**要点**：
- Phalanx armor 可以吸收跳入攻击后反击，是 Marisa 最独特的对空方式
- 但对空跳（空跳后投）无效，需要读心
- OD Tonitrus 有无敌但消耗 2 格 Drive

### 练习2：确认连招训练

**设置**：Dummy 设为随机防御 (Block Random)

**练习内容**：
1. 2LK > 2LP > 确认命中 → xx 236LP (Gladius)
2. 2LK > 2LP > 确认防御 → 停手
3. 2MP > 5MP > 确认命中 → xx 236LP
4. 5HP > 确认命中 → xx 236MK > 236HP

**目标**：命中确认准确率 70% 以上

**要点**：
- Marisa 的重攻击慢但伤害高，确认窗口相对宽裕
- 轻确认 (2LK > 2LP) 用于近距离
- 中确认 (2MP > 5MP) 是最常用的中距离确认

### 练习3：惩罚训练

**设置**：录制 Dummy 出各种可惩罚招式

**练习内容**：
1. Dummy 升龙被防 → (PC) 5HK > 5HP xx 236MK > 236HP
2. Dummy DI 被防 → 5HP xx 236MK > 236HP
3. Dummy HP 头槌被防 → 5HP xx 236MK > 236HP
4. Dummy 超杀被防 → 最大惩罚连招

**目标**：大破绽 100% 出最大惩罚

**要点**：
- Marisa 的惩罚伤害全游戏最高之一，必须抓住每一次机会
- PC 起手 5HK 浮空后可以接更长连招
- 不要用轻攻击惩罚大破绽，永远追求最大伤害

### 练习4：起攻/压制训练

**设置**：Dummy 设为起身后随机行动

**练习内容**：
1. 角落击倒后 → 236[HP] (蓄力 Gladius) meaty
   - 对手防 → Gladius 命中后继续压制
   - 对手按键 → armor 吸收后 Gladius 命中
   - 对手 DP → armor 吸收后 Gladius 命中（!）
2. 中场击倒后 → 前走 meaty 5MP → 帧陷阱
3. 投后 → 前走 meaty 2MP → 投/打二择

**目标**：角落蓄力 Gladius meaty 10/10 时机正确

**要点**：
- 蓄力 Gladius meaty 是 Marisa 最强的起攻手段
- 对手 DP 都会被 armor 吸收后反击，极其恐怖
- 但被投技破解 — 如果对手读到了蓄力会用投技打断
- 所以需要混合使用投技/shimmy 来对应

### 练习5：角色专属技能训练 — Armor 管理与蓄力

**Scutum (214K) 派生训练**：
```
步骤1：Scutum 吸收后派生 P
Dummy 出 5MP → 你 214K 吸收 → P 反击
练到 10/10 反击命中

步骤2：Scutum 吸收后派生 K
Dummy 出 5MP → 你 214K 吸收 → K 前冲 → 投/打择
练到前冲后能接投技或打击

步骤3：Scutum 对飞道具
Dummy 出飞道具 → 214K 吸收 → K 前冲接近
```

**蓄力 Gladius 练习**：
```
步骤1：蓄力时间掌握
236[HP] 按住 → 看到 armor 闪光 → 松开
练到能稳定蓄满

步骤2：蓄力 Gladius meaty 时机
击倒 Dummy → 蓄力 Gladius → 刚好在起身帧命中
从不同击倒技开始练（投后/Gladius 后/扫腿后）

步骤3：蓄力 vs 不蓄力的切换
随机选择蓄力/不蓄力 Gladius
练到两种都能流畅输出
```
