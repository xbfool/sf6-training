# Zangief 专属训练指南

## 一、角色定位与核心打法

Zangief 是 SF6 中的纯投技角色，拥有全游戏最高体力 (10500) 和最高单次投技伤害。核心打法只有一个目标：**缩短距离，然后 SPD**。

**核心思路**：
- 全游戏最依赖"接近"的角色，没有飞道具、没有远程手段
- 一旦贴身，投/打/shimmy 三择让对手必须猜拳，猜错就吃巨额伤害
- SPD (360+P) 是灵魂招式，所有战术围绕它展开
- Double Lariat (214P) 上半身打击无敌，是中距离博弈的关键招式
- SA3 Bolshoi Storm (720+P) 是全游戏最高单次伤害超杀

**距离策略**：
- 全屏：前走 + Drive Parry 弹反飞道具，耐心接近
- 中距离：2MK / 5MP 牵制，寻找 DR 接近或 Lariat 突破机会
- 近距离：投/打/shimmy 三择地狱开始

## 二、核心连招

来源: [SuperCombo Wiki - SF6 Zangief Combos](https://srk.shib.live/w/Street_Fighter_6/Zangief/Combos)

**指令说明**：
- PP = Double Lariat (旋风, Zangief唯一的打击必杀技)
- PPP = OD Double Lariat
- 360P = SPD (螺旋打桩机)
- 360K = Siberian Express
- 360KK = OD Siberian Express
- j.360K / j.360KK = 空中 Siberian Express
- 3MP = 斜前中拳 (命令技, 核心连招中继!)
- 6HP = 前重拳 (命令技)
- 6HK = 前重脚 (命令技)
- 3HK = 斜前重脚 (命令技)
- 6MK = 前中脚 (命令技)
- j.2HP = 空中下重拳
- j.8HP = Headbutt (空中上重拳)
- SA1 = 236236K
- SA2 = 214214P (可按P追加)
- SA3 (CA) = 720P
- tk = tiger knee (最低空输入)

**关键说明**：
- 3MP (斜前中拳) 是 Zangief 连招的核心中继技，几乎所有连招都经过 3MP
- PP (Double Lariat) 是主要连招终结技
- tk j.360KK = tiger knee 最低空 OD Siberian Express，高级技巧

## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心**：同为投技角色的内战，关键在于谁先读对投/打择。

- **距离控制**：中距离用 2MK、5MP 牵制，不要急着贴身
- **Lariat 优势**：214P 上半身无敌可以打掉对手的跳入和部分攻击
- **投技读心**：
  - 读到对手要投 → 后跳 j.HP 惩罚（投技打空硬直极大）
  - 读到对手要打 → 蹲防后 SPD 反击
- **vs Manon 特别注意**：不要让她升 Medal！前期积极进攻，不给她 SA1/360+K 命中的机会
- **vs Lily 特别注意**：注意她的 Condor Spire 突进距离，蹲防后用 5LP 打断

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心**：稳住防守，用体力优势换取接近机会。

- **防守优先**：Zangief 体力 10500 全游戏最高，被打几下不慌
- **对空关键**：
  - 214P (Lariat) 上半身无敌，是对付跳入的利器
  - 63214K (Tundra Storm) 对空投，命中伤害极高
  - 速攻角色跳多，对空成功一次 = 他们失去一大截血
- **被压制时**：Drive Reversal (防中 6+HP+HK) 花 2 格推开，比乱动安全
- **接近方法**：
  - 前走 + 蹲防交替推进
  - 2MK xx DR 接近后施压
  - DI 在对手连续压制时可以翻转局面
- **注意**：不要被 Cammy 的 Quick Spin Knuckle 反复突进拉扯，保持蹲防等她落地后惩罚

### vs 远程角色 (JP/Guile/Dhalsim)

**核心**：这是 Zangief 最困难的对局，需要极度耐心。

- **接近方法**：
  - 全屏：前走 + Drive Parry 弹反飞道具（恢复 Drive Gauge）
  - 中距离：Drive Rush 穿过飞道具间隔
  - DI 可以吸收 2 次飞道具继续前进（但距离近才有效）
- **具体对策**：
  - vs JP：注意他的 236P 飞道具和 214P 陷阱，从上方跳入要小心被 623P 对空
  - vs Guile：Sonic Boom 间隔 Drive Parry + DR 接近，不要在中距离跳（Flash Kick 对空）
  - vs Dhalsim：前走推进，注意他的远距离 5HP/2HP 牵制；贴身后不要让他传送逃跑
- **贴身后策略**：一旦接近绝不轻易放手，投/打/shimmy 猛攻，不给对手重新拉开距离的机会
- **心理准备**：这类对局可能需要很长时间才能接近，保持耐心，不要恐慌跳入被对空

### vs 重型角色 (Marisa/Honda)

**核心**：利用 Zangief 的投技优势破解对方的 armor 和防守。

- **vs Marisa**：
  - SPD 投技直接破解她的 armor（投技无视 armor）
  - 不要和她比 armor — Zangief 没有 armor 招式
  - 蓄力 Gladius 读心 → 跳过去惩罚 或 SPD 直接投
  - Scutum (214K) 看到后不要出招，等她派生再反应
  - 中距离 2MK 牵制，接近后 SPD 三择
- **vs Honda**：
  - LP Headbutt (-2) 基本安全但可以抢帧尝试 SPD
  - HP Headbutt 被防后大惩罚：5HP > 6HP xx 632146P
  - 百裂张手压制时用 Drive Reversal 推开
  - Oicho 是蓄力投，对方需要蓄力 ← 才能出，利用这个限制接近

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心**：最经典的投技 vs 波动拳对决，围绕飞道具接近和近身择展开。

- **反飞道具**：
  - Drive Parry 弹反波动拳 + DR 前冲接近
  - 全屏安全跳跃越过飞道具
  - 214P (Lariat) 可以打掉部分飞道具（距离要对）
  - DI 吸收飞道具接近（中距离有效）
- **注意升龙**：
  - 贴身时不要无脑跳，对方升龙等着你
  - 读到升龙 → 蹲防 → 升龙被防后最大惩罚连招
  - vs Ken：注意他的 DR 接近和 OD 升龙无敌
  - vs Akuma：体力低但爆发高，一旦贴身用投/打择快速击杀
  - vs Luke：注意他的 5MP/2MP 牵制范围，不要在他的最佳距离打
  - vs Ryu：最传统对局，耐心接近后猛攻

## 四、训练模式练习法

### 练习1：对空反应训练

**设置**：训练模式 > Dummy 设为随机跳入 (Record: 前跳 j.HP / 前跳空跳 / 不跳)

**练习内容**：
1. 中距离站立，Dummy 随机跳
2. 看到跳入 → 214P (Lariat) 对空
3. 看到前跳 → 63214K (Tundra Storm) 对空投
4. 没跳 → 继续牵制

**目标**：连续 10 次对空成功率 80% 以上

**要点**：
- Lariat 上半身无敌，是 Zangief 最可靠的对空
- Tundra Storm 对空投伤害更高但需要更精确的距离
- 练到"看到跳就自动出 Lariat"的肌肉记忆

### 练习2：确认连招训练

**设置**：Dummy 设为随机防御 (Block Random)

**练习内容**：
1. 2LP > 2LP > 确认命中 → 2LK xx 236236K (SA1)
2. 2LP > 2LP > 确认防御 → 不出招 / 走上去投
3. 2MP > 5MP > 确认命中 → xx 214P (Lariat)
4. 2MP > 5MP > 确认防御 → 停手 / tick throw

**目标**：命中确认准确率 70% 以上

**要点**：
- 轻攻击确认有约 15-16 帧窗口（2LP > 2LP 之间）
- 看到命中火花才接后续，看到防御盾则停手
- 先练慢速确认，再逐步加速

### 练习3：惩罚训练

**设置**：录制 Dummy 出各种可惩罚招式

**练习内容**：
1. Dummy 出升龙 (被防) → 5HP > 6HP xx 632146P (最大惩罚)
2. Dummy 出 DI (被防) → 5HP > 6HP xx 632146P
3. Dummy 出 HP 头槌 (Honda) → 5HP > 6HP xx 必杀技
4. Dummy 出超杀 (被防) → 根据帧数选择惩罚

**目标**：看到大破绽 100% 出最大惩罚连招

**要点**：
- 升龙被防是最常见的大惩罚机会 (-47帧)，必须练到肌肉记忆
- 不要慌张出轻攻击，冷静出 5HP 开始最大伤害

### 练习4：起攻/压制训练

**设置**：Dummy 设为起身后随机行动 (随机: 防御/按键/跳/投)

**练习内容**：
1. SPD 命中后 → 前走 meaty 2LP → 对手防 → tick throw SPD
2. SPD 命中后 → 前走 meaty 2LP → 对手跳 → 对空 Lariat
3. Borscht 命中后 → 前走 meaty 5MP → 帧陷阱 → Counter Hit 连招
4. 角落击倒后 → DR meaty → 投/打二择

**目标**：每种起身选项都有对应的应对方案

**要点**：
- 击倒后的走位很重要，要刚好在 meaty 攻击的距离
- tick throw 是 Zangief 的灵魂，2LP 后缓冲 360 必须熟练

### 练习5：角色专属技能训练 — 360/720 输入

**Hitbox 360 输入训练**：

```
步骤1：原地 SPD
站立状态 → 6 > 2 > 4 > 8+P
练到 10/10 成功

步骤2：蹲防起始 SPD（推荐方法）
蹲防(↓+←) → 松开↓ → 按→ → 按↑+P
即: 1(下后) > 4(后) > 6(前) > 8+P(上+拳)
练到 10/10 成功，不跳起

步骤3：Tick Throw SPD
2LP(出招时按着↓) → 恢复帧中: 6 > 8 > 4 + P
练到连续 5 次不失误

步骤4：DR SPD
2MK xx DR(66) → 冲刺中: ↓ > ← > ↑ + P
利用 DR 前冲动画隐藏 360 输入
练到连续 5 次不失误

步骤5：720 超投（前冲缓冲法）
66(前冲) → 冲刺中: ↓←↑→↓←↑+P
注意：↑(跳跃键)必须极快点按释放（POP操作）
先练到能出就行，再练稳定性
```

**每日 360 热手流程**：
1. 原地 SPD x10
2. 蹲防 SPD x10
3. Tick SPD x10
4. DR SPD x10
5. 720 x5 (出了就行)
