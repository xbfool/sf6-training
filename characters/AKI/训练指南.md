# AKI 专属训练指南

## 一、角色定位与核心打法

AKI 是 SF6 中的毒系角色，远程控制 + 设陷型打法。通过毒液持续伤害（DOT）+ 毒水洼控制空间 + 蛇形姿态（Snake Step）低姿态接近，构成非传统的消耗打法。

**核心思路**: 不追求一击必杀，而是通过毒液积累总伤害。在远距离用 5HP 超远牵制 + Nightshade Pulse（236P）留毒洼 → 击倒后 236P 毒洼覆盖起身位置 → 对手踩毒持续掉血 → Snake Step（214K）低姿态蛇步接近出招 → 毒积累到致命程度。

**最佳距离**: 中远距离（5HP 牵制范围 + 236P 毒洼设置范围）。近距离有 Snake Step 派生但防御薄弱（无无敌技）。

**关键招式总览**:

| 招式 | 指令 | 用途 |
|------|------|------|
| Nightshade Pulse | 236P | 毒液攻击 + 留毒洼 |
| Cruel Fate | 214P | 毒爪突刺 |
| Snake Step | 214K | 低姿态蛇步 + 派生 |
| Sinister Slide | 236K | 低姿态滑行 |
| Orchid Spring | 623P | 对空毒喷 |
| SA1 | 236236P | 大毒液超杀 |
| SA3 | 236236K | Level 3 超杀 |

**毒系统**:
- 多个必杀技命中后给对手中毒状态（持续掉血 DOT）
- Nightshade Pulse（236P）命中后地面留下毒水洼，对手踩到中毒
- 毒伤害持续累积，不需要一次打出大伤害
- 核心思路：让对手始终处于中毒状态，毒 + 普通伤害合计致命

## 二、核心连招

### BnB连招（必练）

```
2MK xx 236LP                          蹲中脚接轻毒液，安全确认 + 留毒洼
2MP > 5MP xx 214MP                     中拳确认接毒爪，中毒
5HP(远距离牵制) 单发打点               超远距离牵制，AKI 最强普通技
2LK > 2LP xx 236LP                     轻确认接毒液
```

### Drive Rush连招

```
2MK xx DR > 2MP > 5MP > 5HP xx 214HP   标准DR高伤害 + 中毒
2MK xx DR > 2MP > 5MP xx 236MP          DR接中毒液留毒洼
2MK xx DR > 5MP > 5HP xx 236HP          DR接重毒液
```

### Snake Step 连招（角色特色）

```
214MK(Snake Step) > P > 2MP > 5MP xx 214MP   蛇步派生拳接连招
214MK(Snake Step) > K                          蛇步派生脚（低打击）
214MK(Snake Step) > 投                         蛇步接投技
214MK(Snake Step) > P > 2MK xx DR > 5HP xx 214HP  蛇步DR延伸
```

### 惩罚连招

```
对手DP被防:
5HP > 5MP > 2MP xx 214HP               重拳起手最大伤害 + 中毒
5HP xx DR > 2MP > 5MP > 5HP xx 214HP   DR延伸惩罚

DI被防:
5HP xx DR > 5MP > 5HP xx 214HP > SA1   超杀收尾

对手投技打空:
5HP > 5MP xx 214HP                      重拳惩罚 + 中毒
```

### 跳入连招

```
j.HP > 5HP xx 214MP                     跳入接毒爪
j.HP > 2MP > 5MP xx 236MP               跳入接毒液留洼
j.HP > 2MK xx DR > 5MP > 5HP xx 214HP   跳入DR最大伤害
```

### 超杀收尾


## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心**: 远距离 5HP 牵制 + 毒洼阻止接近。AKI 对投技角色有一定优势。

- **远距离**: 5HP 超远牵制不断打点，236P 留毒洼在地面阻止前进
- **对手前走接近**: 毒洼迫使对手绕行或踩毒，减慢接近速度
- **对手跳入**: 623P（Orchid Spring）对空毒喷
- **Snake Step**: 214K 低姿态可以回避部分高打点的攻击，派生出招骚扰
- **被贴身**: AKI 没有无敌技！Drive Reversal 是唯一安全脱身方式
- **注意**: 被 Zangief SPD/Manon 投到后伤害巨大，绝对要保持距离

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心**: 毒洼控制空间减缓速攻角色的接近速度。防守端缺乏无敌技是大弱点。

- **中远距离**: 5HP 牵制 + 236P 毒洼设置在对手前进路线
- **对空**: 623P（Orchid Spring）对空，但注意该技没有无敌，需要提前出
- **Snake Step**: 214K 低姿态可以回避部分跳入和高打点攻击
- **被压制**: Drive Reversal 脱压是最重要的防守手段，AKI 没有 DP
- **毒积累**: 即使被压制，之前的毒伤害持续生效，不要恐慌
- **注意**: AKI 防御最薄弱，被速攻角色贴身压制时只能靠 Drive Reversal 和读心

### vs 远程角色 (JP/Guile/Dhalsim)

**核心**: 远程对轰，AKI 的 5HP 牵制范围极远，毒洼可以限制对手的行动空间。

- **vs JP**: 5HP 可以在远距离打到 JP；236P 毒洼可以阻止 JP 的 Torbalan 地面灵力覆盖。注意 JP 的传送可能绕过毒洼
- **vs Guile**: Sonic Boom 频率高，用 Drive Parry 弹反接近或 236K（Sinister Slide）低姿态穿过。5HP 打点牵制
- **vs Dhalsim**: 远距离 5HP 对决，两个角色都在最远距离打点。236P 毒洼控制地面。注意 Dhalsim 的传送可能绕过毒洼
- **通用**: AKI 的毒是持续伤害，远程对轰中毒积累的总伤害很可观

### vs 重型角色 (Marisa/Honda)

**核心**: 远距离毒积累消耗，不要正面硬碰硬。

- **远距离**: 5HP 牵制 + 236P 毒洼设置，持续消耗
- **毒洼控制**: 重型角色移动慢，毒洼限制前进路线效果好
- **Honda 头槌**: 预判用 5HP 打断或 236P 留洼阻止
- **Marisa armor**: 注意毒液被 armor 吸收时仍然能中毒（部分毒效果穿 armor）
- **Snake Step**: 214K 低姿态回避高打点 armor 攻击
- **注意**: 被贴身后无无敌技，Drive Reversal 是生命线

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心**: 5HP 远距离牵制 + 毒洼控制，利用毒积累总伤害优势。

- **远距离**: 5HP 在对手放波动拳时打到（AKI 的 5HP 范围极远）
- **236P 毒洼**: 设置在中距离，限制对手的放弹和前进空间
- **Drive Parry**: 弹反飞道具后 DR 接近或 236K 低姿态滑行接近
- **Snake Step 接近**: 214K 低姿态穿过飞道具（部分情况下可以）
- **中距离**: 2MK xx 236LP 安全打点 + 留毒洼
- **注意**: 这些角色都有无敌升龙，不要在近距离随意出慢招或跳入

## 四、训练模式练习法

### 练习1：对空反应训练

**目标**: 稳定用 623P（Orchid Spring）对空

**设置方法**:
1. 训练模式 > Dummy 设置为"随机跳入"
2. AKI 站在中远距离
3. Dummy 跳入时反应出 623MP

**练习要点**:
- 623P 没有无敌，需要提前出招（预判性对空）
- 不同距离选择不同版本
- 无法对空时：后跳回避 或 214K（Snake Step）低姿态回避
- OD版（623PP）属性更好

**注意**: AKI 的对空是弱项，需要更多练习来弥补。对空不稳定时用后跳保命。

**通过标准**: 连续10次随机跳入中成功应对 7 次以上

### 练习2：确认连招训练

**目标**: 2MK 确认接 236LP（毒液 + 留洼）

**设置方法**:
1. Dummy 设置为"随机防御"
2. 出 2MK，命中时取消接 236LP
3. 被防时不取消或只接安全的 236LP（被防也留毒洼）

**练习要点**:
- 2MK 命中 → 236LP（留毒洼，安全）
- 2MK 命中 → DR > 2MP > 5MP > 5HP xx 214HP（DR延伸高伤害）
- 2MK 被防 → 236LP 也相对安全（留毒洼控制空间）

**通过标准**: 30次中正确确认 25 次以上

### 练习3：惩罚训练

**目标**: 对手可惩罚招式后最大伤害 + 中毒

**设置方法**:
1. Dummy 录制：起身后出升龙拳
2. 你防住后 5HP > 5MP > 2MP xx 214HP

**练习要点**:
- 标准惩罚：5HP > 5MP > 2MP xx 214HP（+ 中毒效果）
- DR 惩罚：5HP xx DR > 2MP > 5MP > 5HP xx 214HP
- 超杀惩罚：... xx 214HP > SA1

**通过标准**: 连续10次 DP 被防后全部成功惩罚

### 练习4：起攻/压制训练

**目标**: 击倒后 236P 毒洼覆盖起身

**设置方法**:
1. 击倒对手后立刻 236P（毒洼覆盖起身位置）
2. 对手起身踩毒 + 你的投/打二择

**练习要点**:
- **毒洼起攻**: 击倒 > 236LP（毒洼覆盖起身）> 前走 > 投/打二择
- **Snake Step 起攻**: 击倒 > 214K（蛇步接近）> 派生出招/投
- **双重威胁**: 236P 毒洼 + 近身投/打，对手起身踩毒必须应对两层威胁
- **DR meaty**: 击倒 > 236LP 留洼 > DR 贴身 > 投/打
- 不同击倒技后的 236P 设置时机练习

**通过标准**: 连续10次击倒后毒洼覆盖 + 起攻成功

### 练习5：角色专属技能训练 — Snake Step 运用与毒积累

**目标**: 掌握 Snake Step 多变的派生 + 毒积累战术

**训练A — Snake Step 派生**:
1. 214K（Snake Step 低姿态进入）> P（拳派生攻击）
2. 214K > K（脚派生低打击）
3. 214K > 投（蛇步接投技）
4. 214K > 无派生（蛇步后恢复，骚扰/探距离）
5. 练习在不同距离选择不同派生

**训练B — 远距离 5HP 牵制**:
1. 5HP 最远距离打点 x 20（感受最大范围）
2. 在对手出招的恢复帧用 5HP 空振惩罚
3. 5HP 打中后的节奏（不需要取消，收招后准备下一步）

**训练C — 毒洼布局**:
1. 236LP（近距离毒洼）/ 236MP（中距离）/ 236HP（远距离）
2. 练习在不同位置设置毒洼控制空间
3. 对手进入毒洼范围时的追击方案
4. 击倒后 236P 精确覆盖对手起身位置

**训练D — 毒积累意识**:
1. 对 Dummy 持续施加毒效果，观察毒伤害累计
2. 练习在对手中毒状态下的打法调整（更保守，让毒自然掉血）
3. 毒 + 普通攻击的伤害叠加时机

**通过标准**: Snake Step 4种派生各 10 次无失误，毒洼覆盖位置准确 8/10

## 六、每日练习模板

```
总时间: 30-40 分钟

【热身】5 分钟
- 5HP x 15（远距离牵制，感受最远距离，AKI 最重要的普通技）
- 236LP/MP/HP x 5 each（毒液三个版本+毒洼位置）
- 214K > P/K/投 x 5 each（Snake Step 三种派生）
- 623MP x 10（Orchid Spring 对空手感）

【连招练习】10 分钟
- 2MK xx 236LP x 10（基础确认+留毒洼）
- 2MP > 5MP xx 214MP x 10（中拳确认+中毒）
- 214MK > P > 2MP > 5MP xx 214MP x 10（Snake Step 连招，重点）
- 2MK xx DR > 2MP > 5MP > 5HP xx 214HP x 5（DR 高伤害）
- 超杀收尾 x 5

【确认训练】5 分钟
- Dummy 随机防御，2MK 确认 x 20
  - 命中 → 236LP 或 DR 连招
  - 被防 → 236LP（安全留洼）

【对空训练】5 分钟
- Dummy 随机跳入，623P 对空 x 15
- 对空不及时 → 后跳回避 x 5
- 目标: 成功率 70%+（AKI 对空较弱，70% 是合理目标）

【毒洼起攻训练】10 分钟
- 击倒后 236LP 毒洼覆盖 x 10
- 毒洼 + 投/打二择 x 10
- Snake Step 起攻变化 x 5
- 远距离毒洼空间控制 x 5

【实战应用】5-10 分钟
- Rank 或 Casual 实战 2-3 局
- 专注: 毒洼设置频率、5HP 牵制命中率、Snake Step 运用
- 记录: 毒洼设置次数、对手中毒时间占比、被近身压制次数
```
