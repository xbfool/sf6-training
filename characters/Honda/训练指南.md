# Honda 专属训练指南

## 一、角色定位与核心打法

Honda 是 SF6 中的相扑蓄力角色，以 Headbutt 头槌突进为核心，百裂张手压制力强。在 Hitbox 控制器上，蓄力角色天然拥有巨大优势（SOCD 蓄力技巧）。

**核心思路**：
- Headbutt ([4]6P) 是核心突进招式：LP 版 -2 基本安全，HP 版高伤害但不安全
- 百裂张手 (236P) 是近身压制核心，OD 版 +4 命中后直接投/打二择
- Sumo Smash ([2]8K) 上升下砸，兼顾对空和改变轨迹
- Oicho ([4]6K) 蓄力指令投，近身二择工具
- 蓄力角色的特点：按住方向蓄力 → 松开+反方向出招，需要提前准备

**距离策略**：
- 全屏：蓄力准备，LP Headbutt 安全突进或百裂张手牵制
- 中距离：Honda 的最佳距离！Headbutt 突进 / 2MK 牵制 / 前走接近
- 近距离：百裂张手压制 + Oicho 投技二择

**Hitbox 蓄力优势**：
- 按钮蓄力比摇杆更精确，蓄力时间最小化
- SOCD 技巧允许"边走边蓄力"和"瞬间释放蓄力技"
- Sumo Smash 可以用 SOCD 实现"低姿态对空"（按住↓保持蹲姿到最后一刻）

## 二、核心连招

### BnB连招（必练）

```
1. 2LP > 2LP > 5LP xx [4]6LP (LP Headbutt)
   用途：轻攻击蓄力确认
   说明：轻攻击串给你时间蓄力，确认后接安全头槌
   蓄力技巧：出 2LP 时已经在按↓+←蓄力

2. 5MP > 5MP xx 236MP (百裂张手 HHS)
   用途：中拳接百裂，中距离主力
   说明：5MP 按住 ← 蓄力，同时用中拳牵制

3. 2MK xx [4]6MP (MP Headbutt)
   用途：蹲中脚接头槌
   说明：2MK 时已经蓄力 ←，确认后直接 6+P
```

### Drive Rush连招

```
1. 2MK xx DR > 5HP xx 236HP (HHS) > [4]6HP (HP Headbutt)
   用途：DR 百裂接头槌，高伤害路线
   说明：DR 后用 5HP 起手追求最大伤害

2. DI (hit) > 5HP xx OD236P (OD HHS) > [4]6HP
   用途：DI 命中后高伤害
   说明：OD 百裂后接 HP 头槌

3. 2MK xx DR > 5HP xx 236HP > [4]6HP xx SA3
   用途：满槽最大伤害
   说明：百裂头槌接 SA3 收尾
```

### 惩罚连招

```
1. 5HP xx 236HP (HHS) > [4]6HP (HP Headbutt)
   用途：标准惩罚，百裂接高伤头槌
   说明：大破绽时用 5HP 起手

2. 5HP xx OD236P (OD HHS) > [4]6HP > SA1
   用途：有超杀时的高伤害惩罚
   说明：OD 百裂给更多追击时间

3. (起身后) 5LP > 5LP xx [4]6LP
   用途：小破绽快速惩罚
   说明：-4~-5 的破绽用轻攻击起手
```

### 跳入连招

```
1. j.HP > 5HP xx 236HP (HHS) > [4]6HP
   用途：跳入百裂头槌
   说明：跳入命中后的标准高伤害路线

2. j.HP > 5HP xx 236HP > [4]6HP xx SA3
   用途：跳入满槽最大伤害
```

### 超杀收尾

```
1. ...[4]6HP xx [4]6[4]6P (SA1 超级头槌)
   用途：头槌接超级头槌
   说明：蓄力超杀需要提前准备好蓄力
   输入：头槌出完后快速 ← > → > ← > → + P

2. ...[2]8[2]8K (SA2 多段踩踏)
   用途：有无敌的超杀
   说明：垂直蓄力超杀，对空或反击用

3. ...xx 236236K (SA3)
   用途：Level 3 最大伤害收尾
   场景：满 3 槽时的终极收尾
```

## 三、对各类角色对策

### vs 投技角色 (Zangief/Manon/Lily)

**核心**：利用 Headbutt 突进的机动性优势，不要被贴身。

- **距离控制**：
  - LP Headbutt (-2) 安全突进后立刻后退，不给对手投技距离
  - 百裂张手在中距离打点，推对手回去
- **Oicho 指令投**：Honda 自己也有蓄力指令投 ([4]6K)，近身可以投/打二择
- **vs Zangief**：
  - 不要在贴身和他比投技，他的 SPD 伤害更高
  - 中距离 Headbutt 突进打点后拉开
  - Sumo Smash ([2]8K) 可以用来改变轨迹避开 SPD 范围
- **vs Manon**：
  - 不让她升 Medal！用 Headbutt 保持距离打点
  - 她 Medal 低时投技伤害不高，不用太怕被投
- **vs Lily**：百裂张手压制力比她的牵制强，控制中距离

### vs 速攻角色 (Cammy/Kimberly/Rashid/Ken)

**核心**：利用 Honda 的高血量和百裂压制抗住速攻，找机会反击。

- **防守策略**：
  - 蹲防为主（蹲防时可以蓄力 [2] 和 [4]）
  - 被压制时 Sumo Smash ([2]8K) 可以对空/打断跳入
  - Drive Reversal 推开
- **百裂压制**：
  - 速攻角色贴身时 236P 百裂张手推开
  - OD 百裂 (+4) 命中后直接投/打二择
- **对空关键**：
  - [2]8K (Sumo Smash) 是蓄力对空
  - 蹲防时已经蓄力 [2]，看到跳按 8+K 即可
  - Hitbox SOCD 优势：按住↓蹲防 → 拇指按↑+K → 对空发动，保持蹲姿到最后一刻
- **注意**：
  - 不要被 Cammy 的速度带节奏
  - Ken 的 DR 接近要小心，准备好蹲防后 Sumo Smash 对空

### vs 远程角色 (JP/Guile/Dhalsim)

**核心**：Headbutt 突进可以穿过飞道具！这是 Honda 对远程角色的核心优势。

- **Headbutt 突进**：
  - LP Headbutt [4]6LP：安全突进穿飞道具，-2 基本不会被惩罚
  - OD Headbutt [4]6PP：快速安全穿过飞道具
  - 在远距离蓄力准备好，看到飞道具立刻 Headbutt 穿过去
- **具体对策**：
  - vs JP：Headbutt 穿过飞道具接近，注意地面陷阱
  - vs Guile：Sonic Boom 间隔 Headbutt 突进；不要中距离跳（Flash Kick 对空）
  - vs Dhalsim：Headbutt 穿过远距离攻击，贴身后百裂压制
- **接近后**：
  - 百裂张手压制 + Oicho 投技二择
  - 不给对手重新拉开距离的机会
  - 角落百裂压制极其恐怖
- **蓄力准备**：
  - 全屏时按住 ← 蓄力等待，看到飞道具发射动画 → 6+P 头槌穿过
  - Hitbox 蓄力优势让这个反应更快更精确

### vs 重型角色 (Marisa/Honda)

**核心**：重型内战拼效率和读心。

- **vs Marisa**：
  - 不要和她的 armor 硬碰硬
  - Headbutt 突进打点后拉开，不要停在她面前
  - 蓄力 Gladius 读心 → Sumo Smash 改轨迹躲避 或 蹲防后惩罚
  - 百裂张手多段攻击可以破解她的 armor（多段打满 armor 后还有攻击判定）
  - Oicho 投技直接无视 armor
- **vs 镜像 (Honda)**：
  - 头槌对轰看版本选择：LP 安全 vs HP 高伤
  - 谁的蓄力管理更好谁占优
  - 百裂张手互相压制，看谁先创造投技机会

### vs 波动拳系 (Ryu/Ken/Luke/Akuma)

**核心**：Headbutt 穿飞道具是 Honda 的天然优势，经典相扑 vs 波动拳对决。

- **反飞道具**：
  - LP Headbutt [4]6LP 穿过波动拳（-2 安全）
  - OD Headbutt 更快更安全
  - 远距离蓄力等待，读心飞道具 → 头槌穿过
  - Drive Parry 弹反飞道具也是选项
- **注意升龙**：
  - 对方读到你的 Headbutt → 可能出升龙反击
  - 不要太有规律地出 Headbutt，混合前走/跳入/百裂牵制
  - 读到升龙 → 蹲防 → 最大惩罚
- **具体注意**：
  - vs Ryu：最传统对局，Headbutt 穿波动拳是核心
  - vs Ken：他的 DR 接近要注意，Sumo Smash 对空
  - vs Luke：他的飞道具恢复快，不要每次都 Headbutt（可能被读心）
  - vs Akuma：血量低，头槌穿飞道具 + 百裂压制可以快速击杀

## 四、训练模式练习法

### 练习1：对空反应训练

**设置**：Dummy 设为随机跳入 (Record: 前跳 j.HP / 前跳空跳 / 不跳)

**练习内容**：
1. 蹲防状态（已蓄力 [2]），Dummy 随机跳
2. 看到跳入 → [2]8K (Sumo Smash) 对空
3. 没跳 → 继续蹲防蓄力 / 百裂牵制

**目标**：连续 10 次对空成功率 80% 以上

**要点**：
- 蹲防时已经蓄力 [2]，看到跳直接 8+K
- Hitbox SOCD 优势：按住↓ → 拇指按↑+K，过渡帧被游戏读取为有效 [2]8
- 可以保持蹲姿到最后一刻才对空，反应时间更充裕
- 这是 Honda 在 Hitbox 上最大的优势之一

### 练习2：确认连招训练

**设置**：Dummy 设为随机防御 (Block Random)

**练习内容**：
1. 2LP > 2LP > 确认命中 → 5LP xx [4]6LP (Headbutt)
2. 2LP > 2LP > 确认防御 → 停手或投
3. 5MP > 5MP > 确认命中 → xx 236MP (HHS)
4. 2MK > 确认命中 → xx [4]6MP (Headbutt)

**目标**：命中确认准确率 70% 以上

**要点**：
- 蓄力确认的关键：出招时已经在蓄力
- 2LP 按着 ↓+← → 蓄力 [4] 已完成 → 确认后 6+P 出头槌
- 5MP 不需要方向输入 → 可以同时按住 ← 蓄力
- 这就是"边出招边蓄力"的核心技巧

### 练习3：惩罚训练

**设置**：录制 Dummy 出各种可惩罚招式

**练习内容**：
1. Dummy 升龙被防 → 5HP xx 236HP (HHS) > [4]6HP 最大惩罚
2. Dummy DI 被防 → 5HP xx 236HP > [4]6HP
3. Dummy HP Headbutt 被防 (镜像) → 5HP xx 必杀技
4. Dummy 不安全招式 → 根据帧数选择惩罚

**目标**：大破绽 100% 出最大惩罚连招

**要点**：
- 防住大破绽时快速按 5HP 开始惩罚
- 百裂接头槌是 Honda 的标准惩罚收尾
- 蓄力准备：防御期间按住 ← 蓄力，惩罚时已经蓄满

### 练习4：起攻/压制训练

**设置**：Dummy 设为起身后随机行动

**练习内容**：
1. HK Sumo Smash 命中后 → meaty 2LP → 投/打二择
   - 对手防 → Oicho ([4]6K) 投技
   - 对手按键 → 2LP Counter Hit → 确认连招
2. 角落 Oicho 命中后 → 紧贴 → meaty 2LP / 投二择
3. OD 百裂 (+4) 命中后 → 直接投/打二择
4. LP Headbutt (-2) 被防后 → 不是你的回合，防守等待

**目标**：每种起攻场景都有正确的后续方案

**要点**：
- HK Sumo Smash 后帧数非常有利，是 Honda 最佳起攻时机
- 角落 Oicho 后紧贴对手，投/打择极强
- OD 百裂 +4 后等于"你的回合"，可以安全进攻

### 练习5：角色专属技能训练 — Hitbox 蓄力技巧

**基础蓄力练习**：
```
步骤1：水平蓄力 Headbutt
按住 ← 至少 40 帧（约 0.67 秒）→ 松开 → 6+P
左右两边各 10 次

步骤2：垂直蓄力 Sumo Smash
按住 ↓ 至少 40 帧 → 松开 → 8+K
左右两边各 10 次
```

**SOCD 蓄力技巧**：
```
步骤3：边走边蓄力
按住 ← 蓄力 → 出 5MP/5HP（不需要方向输入的普通技）
一边蓄力一边牵制
练到 5MP > 5MP xx [4]6P 连贯不断

步骤4：SOCD 蓄力切换
按住 ← → 快速按 → 释放 → 松开 → 回到 ← 蓄力
利用 SOCD 快速在蓄力和中立间切换
练到切换自然流畅

步骤5：SOCD Flash Kick 式 Sumo Smash
按住 ↓+← (蹲防蓄力) → 拇指按 ↑+K
SOCD: ↑+↓ = 中立，但过渡帧读取为有效 [2]8
练到从蹲防直接出 Sumo Smash 10/10
```

**蓄力超杀练习**：
```
步骤6：SA1 ([4]6[4]6P) 蓄力超杀
保持蓄力 ← → 快速: → ← → + P
滑动方法完成双蓄力输入
练到连招中能稳定接 SA1
```

**每日蓄力热手流程**：
1. LP/MP/HP Headbutt 各 5 次
2. LK/MK/HK Sumo Smash 各 5 次
3. Oicho ([4]6K) x5
4. 边走边蓄力 5MP x10
5. SOCD Sumo Smash 对空 x10
