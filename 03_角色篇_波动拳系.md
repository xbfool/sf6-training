# 第三章：角色篇 — 波动拳系 (Shoto)

---

## 1. Ryu (隆)

### 角色定位
均衡型波动拳角色，SF6 的基础教科书。拥有独特的 **电刃 (Denjin)** 蓄力系统。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| 波動拳 Hadoken | 236P | 飞道具，远程牵制 |
| 昇龍拳 Shoryuken | 623P | 无敌对空（LP版无敌1-14帧最多） |
| 龍卷旋風脚 Tatsumaki | 214K | 前进攻击，连招/运角落 |
| **波掌撃 Hashogeki** | **214P** | 近距离能量波，HP版防御+6！可消飞道具 |
| **上段足刀蹴 High Blade Kick** | **236K** | 突进脚，LK角落可Safe Jump，中屏可安全蓄DJ |
| 電刃練氣 Denjin Charge | 22PP | 进入电刃状态，强化波動拳和波掌撃 |
| SA1 真空波動拳 | 236236P | 7帧，打击/投无敌，Burnout最佳反击 |
| SA2 真・波掌撃 | 214214P | 12帧，完全无敌，高近距伤害 |
| SA3 真・昇龍拳 | 236236K | 5帧，全游戏并列最快无敌反击 |

### 推荐 BnB 连招

> 详见 `characters/Ryu/训练指南.md` 完整连招表

**基础确认（日常最常用）：**
```
2MK xx 214MK/236MK              1300/1460伤害 核心牵制确认
  ※ 236MK击倒后可安全蓄电刃！
5MP > 2MP xx 214MK/623HP        2000/2320伤害 中拳确认
2LK > 2LP > 5LP xx 214LK       1190伤害 轻攻击确认
```

**Drive Rush Cancel 连招（核心伤害来源）：**
```
中/重确认 xx DRC > 5HP > 4HP xx 236HK/623HP   2179-3130伤害 主力BnB
  ※ 角落: 4HP xx 236HK > 623MP 更高伤害
轻确认 xx DRC > 5LP > 4HP xx 236HK/623HP      1639/1690伤害
```

**跳入连招：**
```
j.HP > 5HP > 2HP xx 623HP                     基础跳入
j.HP > 5HP > 2HP xx 214HK > 623HP             加龙卷（角落更佳）
```

**惩罚连招（对手DP被防时）：**
```
PC 236HK(足刀) xx SA1                          3560伤害 最简单!不花Drive!
PC 5HK > 5HP > DRC loop xx SA3                 5539-6235伤害 全资源惩罚
```

**电刃状态连招：**
```
DJ 214HP 命中 > PDR 5HP > 214LP > 2HK          2487伤害 1格Drive
DJ 214PP(崩溃!) > 5HP > DRC loop > 623HP xx SA3 全资源最大伤害
```

### 起攻套路
- **623HP 后(+29~38)**: PDR > CH 6MP；角落: 调节5LP > meaty LP波掌(LAUNCH!)
- **2HK 后(HKD+32)**: 前冲 > meaty 5HP；或 meaty HP波掌(DI安全!)
- **2HK 后蓄电刃**: 这是 Ryu 最重要的击倒选择！sweep > 22PP
- **236K 后(+40~46)**: 所有非OD版中屏可安全蓄电刃！角落236LK → **Safe Jump!**
- **前投(+17)角落**: 投循环! 5LP调节时机 > 再投/shimmy

### 电刃系统要点
- 在对手倒地时蓄力 22PP（sweep 后/龙卷后最安全）
- 电刃波动拳变多段，破防能力更强
- 电刃升龙伤害和击飞大幅提升
- 电刃状态有时限，需要合理安排攻势

### 优势
- 工具齐全，任何距离有应对
- 对空 (623HP) 极强
- 电刃系统提供爆发力
- 学习曲线友好

### 劣势
- 机动性一般
- 非电刃时伤害偏低
- 蓄力电刃需要找时机，有风险

## 2. Ken (肯)

### 角色定位
进攻型波动拳角色，核心是 **迅雷脚 (Jinrai Kick)** 派生系统，压制力极强。EVO/CPT 常客，强度一直在线。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| 波动拳 | 236P | 飞道具（恢复比隆慢） |
| 升龙拳 | 623P | 无敌对空 |
| 龙卷旋风脚 | 214K | 前进攻击 |
| 迅雷脚 Jinrai Kick | 236K | 前冲脚+派生 |
| → 低段派生 | 236K > K | 下段踢 |
| → 中段派生 | 236K > 6K | 中段前踢 |
| → 上段派生 | 236K > 4K | 上升踢 |
| SA1 龙尾脚 | 214214K | 突进超杀 |
| SA2 疾风迅雷脚 | 236236K | 多段连击超杀 |
| SA3 升龙烈破 | 236236P | Level 3 超杀 |

### 推荐 BnB 连招

**基础确认：**
```
2MK xx 236LK                         安全确认接迅雷脚
2MP > 5HP xx 623HP                    中拳确认接升龙
5MP > 5MK xx 214MK                    TC接龙卷
```

**Drive Rush 连招（肯的命脉）：**
```
2MK xx 236LK xx DR > 2MP > 5HP xx 623HP      ~3000伤害 核心BnB
5MP > 5HP xx 236MK xx DR > 5MP > 5HP xx 623HP  ~3200伤害
2MK xx 236MK > K xx DR > 5HP > 2HP xx 623HP    迅雷派生DR扩展
```

**跳入连招：**
```
j.HK > 5HP > 2HP xx 623HP             ~2600伤害
j.HK > 5HP > 2HP xx 214HK > 623HP     ~3000伤害 (角落更佳)
j.HK > 5MP > 5HP xx 236MK xx DR > 5HP > 2HP xx 623HP  最大化
```

**角落连招：**
```
(角落) j.HK > 5HP > 2HP xx 214HK > 214HK > 623HP   龙卷x2
(角落) 5HP > 2HP xx 214HK > 5MP > 2HP xx 623HP      角落限定
```

**超杀连招：**
```
2MK xx 236MK xx DR > 5MP > 5HP xx 623HP xx SA2  ~4000伤害
Anti-air 623HP xx SA3                            对空接SA3
```

### 迅雷脚压制（Ken 的灵魂）

```
236MK > 6K（中段派生）→ 命中追击 / 被防+1
236MK > K（低段派生） → 命中接追击
236MK > Emergency Stop → 投技 / 再攻击
```
- 中段/低段/停止投 = 三择，对手很难防
- 被看穿会被大惩罚，需要变化节奏

### 起攻套路
- **623HP 后**: 前步 > meaty 5MP / 投
- **214HK 后**: 距离极近，直接 2LP>2LP>5LK / 投
- **2HK 后**: 前步 > meaty 2MK xx 236K 循环压制
- **迅雷脚循环**: 在对手防守时用 236K 各派生持续施压

### 优势
- 进攻能力极强，迅雷脚派生系统丰富
- DR 连招伤害极高
- 对空依然优秀
- 飞道具辅助节奏控制

### 劣势
- 防守时资源消耗大，容易 Burnout
- 迅雷脚被看穿惩罚很大
- 波动拳恢复帧比隆长
- 过于激进时容易被反打

## 3. Luke (卢克)

### 角色定位
SF6 主角，全面型角色。**Flash Knuckle（闪光拳）** 蓄力系统是核心。几乎没有明显弱点。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| Sand Blast 沙爆弹 | 236P | 飞道具（可蓄力） |
| Rising Upper | 623P | 对空（LP版无敌） |
| Flash Knuckle 闪光拳 | 236K (可蓄力) | 突进拳，蓄力版更强 |
| Avenger 复仇者 | 214P | 前冲+派生 |
| → Avenger > P | 打击派生 |
| → Avenger > K | 低段派生 |
| SA1 Vulcan Blast | 236236P | 多段飞道具超杀 |
| SA2 Eraser | 214214P | 突进超杀 |
| SA3 Pale Rider | 236236K | Level 3 超杀 |

### 推荐 BnB 连招

**基础确认：**
```
5MP > 5MP xx 236LK                    简单双中拳接闪光拳
2MK xx 236LK                          下中脚确认
2LP > 2LP > 5LK xx 623LP              轻攻击确认
```

**Drive Rush 连招：**
```
2MK xx 236LK xx DR > 5MP > 5HP xx 623HP    ~3000伤害 主力BnB
5MP > 5MP xx DR > 5MP > 5HP xx 236MK(蓄力)  DR扩展
5HP xx 236MK xx DR > 5MP > 5HP xx 623HP     重拳起手
```

**跳入连招：**
```
j.HP > 5HP > 5HK xx 623HP             ~2600伤害
j.HP > 5HP > 5HK xx 236MK(蓄力) > 5HP xx 623HP  蓄力扩展
```

**蓄力闪光拳连招：**
```
236MK(满蓄力) > 5HP > 5HK xx 623HP    满蓄力命中追击
236HK(蓄力) > DR > 5HP > 2HP xx 623HP  重蓄力路线
```

### 起攻套路
- **623HP 后**: 前步 > meaty 5MP / shimmy
- **236HK 后**: 可直接压起身
- **2HK 后**: meaty 2MK xx 236K 循环
- **Avenger 压制**: 214P 接近 > 打击/投/延迟多层择

### 优势
- 极其全面，无明显弱点
- 5MP、2MK 帧数和判定都优秀
- 蓄力系统提供额外层次
- 入门友好，上限也高

### 劣势
- 没有特别突出的单一强项
- 不用资源时伤害中等
- 飞道具发生慢
- 被贴身缺乏好的脱身手段（非OD）

## 4. Akuma (豪鬼)

### 角色定位
DLC 经典角色，**玻璃大炮**。体力全游戏最低（~9000），但进攻手段极其丰富。**百鬼袭 (Demon Flip)** 和 **瞬獄殺 (Raging Demon)** 是标志。

### 核心招式

| 招式 | 指令 | 用途 |
|------|------|------|
| 豪波動拳 | 236P | 地面飞道具 |
| 斬空波動拳 | 空中 236P | 空中飞道具 |
| 豪昇龍拳 | 623P | 无敌对空 |
| 竜巻旋風脚 | 214K | 旋风脚 |
| 百鬼襲 Demon Flip | 214P+K 或 236K | 空中翻转+派生 |
| → P 派生 | 空中拳击 |
| → K 派生 | 俯冲脚 |
| → 投派生 | 空中投 |
| SA1 滅殺豪波動 | 236236P | 大波动拳 |
| SA2 金剛國裂斬 | 214214K | 有无敌的冲击波 |
| SA3 瞬獄殺 | LP LP 6 LK HP | 经典即死投！不可防御！ |

### 推荐 BnB 连招

**基础确认：**
```
5MP > 5MP xx 214HK                    简单好oki (DaFeetLee推荐)
5HK > 5MK xx 214LK > 2HK             好oki版本
2MK xx 236MP                          蹲中脚确认
```

**Drive Rush 连招：**
```
2MK xx DR > 5MP > 5HP xx 214HK > 623HP       ~3200伤害
5MP > 5HP xx 214HK xx DR > 5MP > 623HP        DR延伸
```

**跳入连招：**
```
j.HP > 5HP > 6HP xx 214MK > 623MP     ~2800伤害
j.HP > 5HP > 6HP xx 214HK > 623HP xx SA1  超杀收尾
```

**Punish Counter：**
```
5HP(PC) > 5HP xx 214HK > DR > 5MP > 623HP  ~3400伤害
crMP(PC) > 5HP xx 214LK > 2HK              2540伤害 (实用)
```

**SA3 瞬獄殺输入：**
```
近距离: 5LP > 5LP > LP LP 6 LK HP
缓冲: 在轻攻击恢复帧中输入 LP 6 LK HP
```
- 不可防御，对手只能跳或无敌回避
- 心理压力巨大，仅凭存在就影响对手行动

### 百鬼袭 (Demon Flip) 应用

```
击倒后 > 百鬼 (236K)：
  → K(俯冲脚): 中段，命中接连招
  → P(拳击): 打击
  → 投派生: 空中投
  → 空落: 落地接地面投/打击

= 四择！对手极难防守
```

### 起攻套路
- **623HP 后**: 百鬼袭(236K)各种派生二择
- **214HK 后 (角落)**: meaty 5MP / 投
- **空中波动拳压制**: j.236P > 落地攻击
- **百鬼翻转**: 击倒后 236K，四种派生制造混乱

### 优势
- 进攻手段极其丰富（地面/空中/飞道具/投）
- 连招伤害极高
- SA3 瞬獄殺心理威慑巨大
- 百鬼袭制造大量二择

### 劣势
- **全游戏最低体力之一**，一套被吃可能直接死
- 犯错代价比任何角色都大
- 操作难度高
- 需要精确的执行力和判断力
